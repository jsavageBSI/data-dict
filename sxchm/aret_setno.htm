Trigger: 

Indexes:
k-jrnl


Description:
The Set Number corresponds directly to the glet.setno. Each time an
individual posting is made to the general ledger for a given journal,
a new Set Number is assigned. Each entry in ARET will create a new
Set Number. Each set of postings to a customer in ARECE will create a
new Set Number. If the user answers no to the OEEPI Option, "#16 Only
Post Detail if Invoice in Suspense", then each invoice will create a
new Set Number. If the user answers yes, the entire journal will
create one set if the suspense account is not hit.
