vaesl.linealtno - Line # Alternate
  
Label: Line # Alternate
Type: integer
Format: zz9
Decimals: ?
Initial: 0
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
The line alternate number indicates the PO, WT or VA line that this
order is tied to. This field is not enterable from VAET but is updated
when the PO/WT/VA line is created from the order (on a drop line or a
tie).
 
Content:
PO/WT/VA line number
