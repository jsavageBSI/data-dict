icsw.surplusty - Surplus Type
  
Label: Surplus Type
Type: character
Format: x(1)
Decimals: ?
Initial: 
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: A)ctual Monthly Usage or I)CSW Usage Rate (Blank Same as I)
Trigger: 



Description:
Default calculation method for surplus. This value is used within the ICRIS
report when specified to calculate surplus values for a given product. It is
defaulted in from the ICSL record when created new using a specific product
line.
 
Content:
&lt;blank&gt; or "I" represents "ICSW Usage Rate" for Surplus calculations
"A" represents "Actual Monthly Usage" (based off the last 12 months)
