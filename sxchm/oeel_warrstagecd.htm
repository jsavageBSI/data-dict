oeel.warrstagecd - Warranty Stage
  
Label: Warranty Stage
Type: integer
Format: 9
Decimals: ?
Initial: 0
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 0-Not Enterable, 1-Entered, 2-Printed, 3-Credit Applied For
Trigger: 



Description:
The warranty stage code is used to track the processing of warranty
products to the vendor. After a return line is entered, the OEROW report
can be run to display lines that need to be placed on POs. Once the PO
has been placed and the PO line tied the the sales line, the warranty
stage is updated. It is also incremented when credit is received from
the vendor.
 
Content:
1 = the warranty line has been entered
2 = the OEROW report has printed for the return line
3 = credit has been applied for (the PO-RM has been entered)
4 = credit has been received (the PO-RM has been received)
