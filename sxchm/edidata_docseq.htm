Description:
Sequence within the transaction set the section was received.
This field is used to allow a document to be created and read back in the sequence it was received in ETCC. An example would be in EDI where a section such as lncomm will be received immediately following the receipt of the item section but there is no data seqment within the EDI specification that ties the lncomm section to that specific line.
It also ensures other data seqments are processed in the correct sequence.
