Description:
The GL sub account number from the transaction being revalued. For APET and ARET, this value is taken from the original transaction. For POEH, this value is determined the same way it is in POEIUG. For APEI transactions, this values comes from the APEIG record. For A:PEBC, this value is relevant SASO unmatched account.
