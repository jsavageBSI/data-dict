oeeh.writeoffamt - Total Write-Off Amt
  
Label: Total Write-Off Amt
Type: decimal
Format: zzzzzzzz9.99-
Decimals: 2
Initial: 0
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
The write off amount on a tendered order. This amount is an internal
field and is not loaded until a tendered order is invoiced. When a
tendered order is back ordered, there is room for addon/tax calculation
rounding issues between the original addon/tax calculations on the -00
order and the calculations on the back order. As a result, the order can
be over or under tendered by pennies. In OEAO, a downpayment write-off
amount can be entered. If this amount is greater than 0, an order will
be considered fully tendered if it has been tendered within range of the
downpayment write-off amount. An ARET write-off transaction will be
generated from OEEPI if this occurs.
