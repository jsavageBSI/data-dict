Description:
Used in the OEET Shopping List to display Qty Available value plus a ?CR?
after the quantity if it was adjusted because of an active reservation.
Since ICSW has the Quantity Reserved loaded for the reservation amount
so no one else can use that quantity, the system take Net Avail and adds
back in the active reservation 'reserved' quantity - showing it as available
stock to the order's customer/shipto.
This is a chracter field because it can have CR at the end or not.
