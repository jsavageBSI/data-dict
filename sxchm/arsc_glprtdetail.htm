arsc.glprtdetail - Print GL Detail
  
Label: Print GL Detail
Type: character
Format: x(1)
Decimals: ?
Initial: 
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
Customer Override to AO setting to Print GL detail lines on Receipts
and Invoices for a Customer.
 
Content:
Valid values: "", N, I, R, B
Blank - Use AO default setting for printing GL data.
N - Neither receipts nor invoices get GL data
R - Receipt print GL data only .
I - Invoice Print GL data Only.
B - Both Receipt and Invoice Print GL data.
