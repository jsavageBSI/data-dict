sasoo.whsegroup - Whse Grp
  
Label: Whse Grp
Type: character
Format: x(4)
Decimals: ?
Initial: 
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
An operator can be assigned to a warehouse group. The user is then limited to working only in those warehouses that are part of the group. It provide more flexiblity then 'Home Warehouse Only' functionality which limits the operator to a single warehouse without having to give the operator security to work in all warehouses.
 
Content:
This field contains a character value that represents a warehouse group defined in SASTT. If the value entered does not match a value defined in SASTT, the user will receive the following error message:
Warehouse Group Not Set Up in System Table - SASTT
If no value is entered, the operator is not limited to working in the warehouses that make up the warehouse group.
