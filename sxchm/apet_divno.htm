Description:
Description: This field is used if you are using full divisionalized
accounting. The system determines if you are using divisionalized
accounting by checking if you have your flag set to "yes" in
System Administrator Setup Company (SASC) as sasc.divno. If this flag
is set to "no" in SASC you will not have access to this field. Before
turning this flag on, contact R&D support or your Distribution
Consultant for proper setup. Enterable: The division number is
enterable for the following type of records:
Invoices - transcd of 0
Rebates - transcd of 2
Misc. Credits - transcd of 5
Manual Payments - transcd of 7.
If you leave this field blank and SASC is flaged for divisionalized
accounting the system will default the division number from APSV. The
division number is stored in APSV as apsv.divno. You can override this
value at the time of APET entry.
Allowable Values: This field is 4 integers in length. The system will
validate the existence of this division number by checking SASTT under
