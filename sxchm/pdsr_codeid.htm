Description:
This field is a concatentation of pdsr.levelcd (with possible values of "1,2,3,4,5"), pdsr.rebatecd (with possible values of "c,p,s" for customer,purchase,sale), and pdsr.dropshipty (with possible values of "w,d" for warehouse,dropship).
All of these fields needed to be in the unique key structure. Due to
progress key size limitations, it was necessary to load the three fields
into one for use in the primary key. Thus possible values for the field
are all permutations of the three: (1,2,3,4,5), (c,p,s),(w,d)
exp: (1cw),(1cd),(1pw),(1pd),(1sw),(1sd),(2cw),....
