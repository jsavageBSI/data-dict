Description:
Indicates that the order needs to be totalled. Totalled means the
following:
the special order discount needs to be calculated
the addons need to be calculated
the taxes need to be calculated
the whole order discount needs to be calculated
the customer on order balance needs to be updated.
Totalling occurs in p-oetot.i
