oeel.kitfl - Kit
  
Label: Kit
Type: logical
Format: yes/no
Decimals: ?
Initial: no
Extent: 0
Mandatory: yes
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
This field indicates that the line is a Build On Demand kit and OEELK records are attached to the line item. It will be set to yes for a Non-Stock kit or a Build On Demand kit. It is important to set correctly so that the price/cost roll will work properly and inventory will be affected for components. The kitfl would be set to no for a Prebuilt kit since OEELK records will not exist.
