Description:
Added with JIRA SXWEB-38283/SXWEB-39524. When the AO field is checked, ARSC – General Ledger – has a new section titled “Income Statement General Ledger Accounts”. In these fields, the default values for this customer can be set. Invoice Processing will post to the income statement accounts by customer, if they are set, otherwise, the current hierarchy to select the accounts for income statement postings will be used.
Usage of this logic requires AOData AR - useARIncomeGLAccts to be checked. This can be found in AO – Customers – Defaults – Customer Setup section with the label “Use Customer Income Statement Accounts”.
 
Content:
Validated the same as all general ledger accounts.
