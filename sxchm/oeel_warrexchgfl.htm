oeel.warrexchgfl - Warranty Exchange
  
Label: Warranty Exchange
Type: logical
Format: yes/no
Decimals: ?
Initial: no
Extent: 0
Mandatory: yes
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
The warranty exchange flag is set on the return and sale lines of a
warranty exchange. This feature can be used when returning a line if the
return type is set up in SASTA as a warranty exchange, which will create
a sales line from the return line with a 0 price and cost.
 
Content:
yes = this return created a warranty sales line or this sales line
was generated from a warranty return
no = no warranty exchange is involved
