sasoo.oedotymrgamtmax - DO Max Hold Margin Amt
  
Label: DO Max Hold Margin Amt
Type: decimal
Format: zzzzzzzz9.99
Decimals: 2
Initial: 0
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: Maximum Margin Amount For DO Type OE Order
Trigger: 



Description:
This field contains the maximum margin value for a DO type Order Entry order. When this operator enters a DO type OE order, the margin based on the ordered amount for that OE order will be compared against the value in this field. If the margin value on the OE order is more than the value in this field, the OE order will be placed on hold.
 
Content:
This field contains a positive numeric value that represents the maximum margin value based on the ordered amount for which this operator can enter a DO type Order Entry order without that OE order being placed on hold.
A value of zero indictates that there is no maximum margin for this operator.
