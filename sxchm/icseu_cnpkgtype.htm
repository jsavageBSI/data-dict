Description:
Package Type. Links to the SASTT Package Type setup table.
The Unit package type becomes the carton package type when processing
the carton (unit) as a shelf to shipment container. The Package Type carries
a 'Carton ID' value for the Carton ID creation for the carton.
If packing the Unit into other cartons, then the Package Type becomes the
item package type to be used for packing restrictions or mixing item, etc
 
Content:
SASTT Package Type or Blank (no specific type)
