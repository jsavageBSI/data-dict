icsw.baseyrcost - LIFO Base Year Cost
  
Label: LIFO Base Year Cost
Type: decimal
Format: zzzzzz9.99999
Decimals: 5
Initial: 0
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
This represents the LIFO cost for this product. Like all costs, it is
in stocking units unless special price/costing is used for this product.
This cost cannot be used for any updating process but is available on
several IC reports, including a comparitive trial balance. The cost is
calculated by the system using ICROV. The option exists to run this
report in updating mode. If that is done this cost will be updated.
