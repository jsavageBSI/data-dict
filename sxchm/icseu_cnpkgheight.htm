icseu.cnpkgheight - Package Height
  
Label: Package Height
Type: decimal
Format: zzzzz9.99999
Decimals: 5
Initial: 0
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
This is used with cncalcpkgsizefl. If using the dimensional values of the
unit to pack cartons with, then need to know what dimension the
'unit' is being measured in.
Height is required if the option is on.
Unit that is measured dimensionally can be shipped as is, or packed into
another carton, but need the dimensions to pack the unit (Case, Box, etc)
 
Content:
Height of the unit
