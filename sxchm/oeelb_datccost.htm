oeelb.datccost - DATC
  
Label: DATC
Type: decimal
Format: zzzzzz9.99999
Decimals: 5
Initial: 0
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
The "destination area transportation charge" for this line item
represented in the selling unit. The field was originally established to
be a datc cost. It has since be expanded (V4.5) to be a line addon
charge. The label of the cost is user defined. If it is defined in ICAO,
this charge can be entered (and will default from ICSW). This field is
used to pass an additional charge to the customer based on the product
sold and how it is sold (IE: if the product was rushed to the customer).
 
Content:
Defaults from ICSW. Also comes in when a line is set to a rush line.
