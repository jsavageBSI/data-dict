Description:
Under a Product Type restriction (ICSPR) record, there can be a number
of specific type restrictions:
CS - City-State (need city with a state - Jamestown, ND or Jamestown, CO)
ZC-Zip Code
CC-Country Code
CN-Customer Number
ST-Customer Ship To
CT-Customer Type
CP-Customer Price Type
TE-Customer Territory (Sales territory)
CL-Certificate/License

CS, ZC, CC, CN, ST, CT, CP, TE, CL
