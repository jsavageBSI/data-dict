pv_user.pdrestrictty - PD Restriction Type
  
Label: PD Restriction Type
Type: integer
Format: 9
Decimals: ?
Initial: 0
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
Pricing records can be created for a warehouse, a division group, a pricing
region or for use throughout the company. This field is used to restict a user
to maintaining specific types of pricing records and price sheets.
 
Content:
0 - No Restrictions (user can create and maintain all types of records)
1 - Restricted to Warehouse (user can only create and maintain records
set up for a warehouse)
2 - Restricted to Region (user can only create and maintain records set up
for a pricing region)
3 - Restricted to Division Group (user can only create and maintain records
set up for a division group)
4 - Restricted to Warehouse and Region (user can only create and maintain
records set up for either a warehouse or a pricing region)
5 - Restricted to Warehouse and Division Group (user can only create and
maintain records set up for either a warehouse or a division group)
6 - Restricted to Region and Division Group (user can only create and
maintain records set up for either a pricing region or division group)
