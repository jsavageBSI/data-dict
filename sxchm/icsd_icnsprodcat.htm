Description:
Default product category that can be used when a non-stock product is
entered on an OE order that is generated from the Storeroom.
The heirarchy that will be followed when determining what product category
should be applied to the line on the order will be:
1. Use the product category passed from Storeroom
2. Use the warehouse default
3. Use the AO default.
 
Content:
A valid product category set up in SASTT
