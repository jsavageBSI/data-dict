icsw.reservety - Reservation Type
  
Label: Reservation Type
Type: character
Format: x(1)
Decimals: ?
Initial: 
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: (D)elay, (R)eceipts, (A)lways or Blank
Trigger: 



Description:
This field changes the timing at which inventory is actually reserved for an order. The reservation type can be assigned as (d)elayed, against (r)eceipts, (a)lways reserve, or be left blank. Each type defines a different level of hierarchical rules that dictate when and if the product will be reserved upon entry.
 
Content:
D = delay reservation
R = against pending receipts
A = always
