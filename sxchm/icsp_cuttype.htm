icsp.cuttype - Cut Type
  
Label: Cut Type
Type: character
Format: x
Decimals: ?
Initial: N
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
This field will indicate whether the product is to be used with the Dimensional
Cut functionality in VA (MSF# 186085), whether it is to be used with the Linear Cut functionality
in SX RF (MSF# 142967), or whether it is not a cut product.
The initial value is that a product is NOT a cut product.
 
Content:
N - Not a cut product
D - Dimensional Cut product
L - Linear Cut producut (SXRF Cut Product)
