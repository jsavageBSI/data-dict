Description:
Vendor Default setting used via the ION procesSupplierInvoice interface.
If this flag is set, then when loading the Terms Type for the Invoice, the
system will look at the following in order:
1. If the Interface sent the EDI Terms code that cross referenced to a
valid SASTT Terms Code, then that Terms Code is used.
2. If the SupplierHeaderInvoice section of data passed in a PO# and
a valid POEH record was found, it uses the poeh.termstype
3. If neither of the top apply, it will use the apsv.termstype as the default
