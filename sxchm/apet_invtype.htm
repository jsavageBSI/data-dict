apet.invtype - Invoice Type
  
Label: Invoice Type
Type: character
Format: x(3)
Decimals: ?
Initial: 
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: Invoice Type - Used by Cash Requirements and Pay Invoices Due
Trigger: 



Description:
This can be used to group common transactions together. All
transactions with the same type can be selected for reporting through
APRC and paying through APEP. All Utility invoices may be grouped as
Invoice Type 'Utl'. When processing APRC and APEP you can specify the system to process only invoices grouped as 'Utl'. Enterable: The
InType can also be added in Accounts Payable Set Up Vendor (APSV). If the field is established at the APSV level the value will default in
APET from APSV. During APET entry the InType can be overriden.
Allowable Values: Up to 3 characters including periods, slashes, dashes etc.
