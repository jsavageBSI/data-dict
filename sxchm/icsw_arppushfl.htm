icsw.arppushfl - Push
  
Label: Push
Type: logical
Format: yes/no
Decimals: ?
Initial: no
Extent: 0
Mandatory: yes
Val Exp: ?
Val Msg: 
Help: Push From Central Whse or Satellite Pulls From Central Whse
Trigger: 

Indexes:
k-arpwhse


Description:
Authorized Replenishment Path Push Flag. This flag is used only if the
the ARP is from another location, not if from vendor. A true value
indicates that the supplying location will push product to the the
receiving location (this branch). A false indicates that the receiving
location (this location) will request product when needed from the
supplying branch. Note that this flag defaults from ICSL if vendor/line
is provided, then from ICSD.
