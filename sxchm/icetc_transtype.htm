icetc.transtype - Type
  
Label: Type
Type: character
Format: x(2)
Decimals: ?
Initial: 
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 

Indexes:
k-custprod k-postdt


Description:
This field indicates the type of transaction. Each transaction type
indicates the type of affect on GL and on IC balances in ICSW.
 
Content:
UN - Unavailable (not used in stock inv calc)
DO - Direct Order (not used in stock inv calc)
NS - Non-Stock (not used in stock inv calc)
RE - Receipt Increase in inventory
RI - Return In Increase in inventory
SA - Stock Adjust Increase in inventory except when icet.module = "WT"
CA - Cost Adj (not used in stock inv calc)
RO - Return Out Decrease in inventory
IN - Invoice Decrease in inventory
