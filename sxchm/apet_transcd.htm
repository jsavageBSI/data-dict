Label: Tran Type

Mandatory: no
Val Exp: ?

Indexes:
k-apinvno k-inq k-novend k-pidjrnl


Description:
This field represents the type of transaction you are processing:
Invoice, Rebate, miscellaneous Credit, Credit Memo, Manual Payment,
Debit Memo, or Reversal. Enterable: This field is enterable by the
user and is required. The system will default a value of 0 into this
field, which represents an invoice type transaction.
 
Content:
0 invoice debit
2 rebate none
3 sched paymt debit
4 void check debit
5 misc credit credit
6 credit memo credit
7 paymt record credit
8 debit memo debit
9 reversal credit(opp 0,1, or 4)
10 batch costing credit
