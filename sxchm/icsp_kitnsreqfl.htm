icsp.kitnsreqfl - Non Stock Component Required
  
Label: Non Stock Component Required
Type: logical
Format: yes/no
Decimals: ?
Initial: no
Extent: 0
Mandatory: yes
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
The Kit Nonstock Required Flag indicates if the Build on Demand Kit
should contain required Non-stock Kit Component(s). While in OE and BP,
the Operator is notified if entry of at lease one nonstock is required.
In OE, the field is check during the entry process. In BP, the field is
checked during the release process.
The user is allowed to set the flag to 'yes' for a Prebuilt Kit, but
Nonstock Components are not allowed on KP Work Orders. This could be a
future enhancement.
 
Content:
yes = At least one Nonstock Kit Component is required for this BOD Kit
no = No Nonstock Kit Component(s) are required for this BOD Kit
