sastn.merchantid - Merchant ID
  
Label: Merchant ID
Type: character
Format: x(4)
Decimals: ?
Initial: 
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: Merchant ID used in Credit Card Settlement Processing
Trigger: 



Description:
This is the Merchant ID that is used during credit card processing and the settlements. If a merchant id is loaded at the Payment Type level in the system, then it is like a system wide default for that payment.
They also have the ability to attach a merchant id to the whse and payment type on the whse via the new ICSDP function. The ICSDP would override the SASTT Merchant ID if loaded for the payment type directly.
If neither SASTT or ICSDP carry a merchant id via the payment type, then works no different than prior to the new field.
