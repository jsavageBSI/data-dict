oeel.commentfl - Comment
  
Label: Comment
Type: logical
Format: yes/no
Decimals: ?
Initial: no
Extent: 0
Mandatory: yes
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
The comment flag for the line item. If a comment exists on the line
(stored in COM based on the order/suffix/line), a "c" will display to the left of the line item on the Order Entry screen.
 
Content:
yes = a comment is attached to the line item
no = there is no comment attached to the line item
