crsb.pospayurl - Positive Pay URL
  
Label: Positive Pay URL
Type: character
Format: x(80)
Decimals: ?
Initial: 
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: Positive Pay Program Connection URL
Trigger: 



Description:
If the bank offers a Postive Pay program and the check data is to be sent
out using CenPOS, this field contains the URL provided by CenPOS to
connect to the web service they provide to brring in check data.
 
Content:
A URL provided by CenPOS
