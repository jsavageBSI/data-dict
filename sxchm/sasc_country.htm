Description:
Company's Address - Country
 
Content:
Any Country Setup in SASTT. No longer determines the type of
taxing for the system.
Taxing is now driven from AODATA file with the recordtype = "TAX" and
filename = "TaxMethodTy" - loaded and set in SASC - Tax screen.
TaxMethodTy: "S" - Sales Use Tax (SUT) - Old 'US' Country
"G" - Goods and Services TAx (GST) - Old 'CA' Country
Opens up the taxing to new available interfaces and types like "V" -VAT,
or "E"-Enterprise Taxware - International taxing
