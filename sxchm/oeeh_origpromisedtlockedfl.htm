oeeh.origpromisedtlockedfl - Orig Promised Locked
  
Label: Orig Promised Locked
Type: logical
Format: yes/no
Decimals: ?
Initial: no
Extent: 0
Mandatory: yes
Val Exp: ?
Val Msg: 
Help: Is the Original Promise Date Locked
Trigger: 



Description:
Used to flag when the original promisedt can be updated. When origpromisedtlockedfl = true no changes can be made to the original promisedt.
This will default as false when an order is created, and set to true in the Order Finish logic as the user is only allowed to change the originalpromisedt when an order is new.
 
Content:
True when the user cannot change the originalpromisedt value
False when the user can change the originalpromisedt value
