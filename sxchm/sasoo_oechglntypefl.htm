sasoo.oechglntypefl - OE Change Line Type
  
Label: OE Change Line Type
Type: logical
Format: yes/no
Decimals: ?
Initial: no
Extent: 0
Mandatory: yes
Val Exp: ?
Val Msg: 
Help: If 'Yes', Allow Changes to OE Line Type
Trigger: 



Description:
If sasoo.oechglntypefl is yes, the operator is allowed to change the line type
for an existing line in OE Order Entry. If your operator ID has permission to
override the line type, you can change a line type without changing the
line number. Lines may be changed to any of the three types: nonstock,
stock, or special; however, lines for order as needed products cannot be
changed from a special to stock type. The default is no.
