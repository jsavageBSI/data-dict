vaesl.intermprodfl - Intermediate Product Flag
  
Label: Intermediate Product Flag
Type: logical
Format: yes/no
Decimals: ?
Initial: no
Extent: 0
Mandatory: yes
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
Intermediate product flag. This field indicates if the value add
transaction inventory component line item is an intermediate product,
from an external or internal processing section. This is needed by the
PO/WT RRAR functions to not select this line item, since the line item
will be a non-stock product. This intermediate line item is only used
to ship the intermediate product to the next processing step.
 
Content:
yes - the inventory component line item is an intermediate product
no - the line item is not an intermediate product
