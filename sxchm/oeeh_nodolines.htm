oeeh.nodolines - # DO Lines
  
Label: # DO Lines
Type: integer
Format: >>9
Decimals: ?
Initial: 0
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
The number of direct order lines on the order. This field is accumulated
for each line with a "d" in the botype. If the number of DO lines on the
order is equal to the number of lines on the order, the order can be set
to shipped stage when filled from back order fill (and the unfilled
lines will back order). If the nodolines is not equal to nolineitem,
then the DO lines will be moved to a new suffix and filled (from back
order fill).
 
Content:
Accumulated number of DO lines.
