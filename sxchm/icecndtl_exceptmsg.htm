icecndtl.exceptmsg - Exception Message
  
Label: Exception Message
Type: character
Format: x(60)
Decimals: ?
Initial: 
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
If an exception is found during cartonization, then the message is stored
on the exception carton.
The Carton ID is the Order#: "o1234-00 - Exception" and the statusty of the
crton record (icecnhdr.statusty) is 'E'.
Exceptions can be deleted once corrected via the ICECN 'Delete Packages'
function. Repack the item manually, or delete the package if one is created
for the order and re-run auto cartonization.
