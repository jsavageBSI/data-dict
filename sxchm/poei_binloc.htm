poei.binloc - Bin Loc
  
Label: Bin Loc
Type: character
Format: xx/xx/xxx/xxx
Decimals: ?
Initial: 
Extent: 2
Mandatory: no
Val Exp: ?
Val Msg: 
Help: Building / Row / Section / Shelf
Trigger: 



Description:
When a nonstock product is recieved in either POEI or WTEI, the user will be able to enter the location into which the nonstock product was received. When the receiving update is run, the bin location stored on the POEI record will be used to update the ICENL bin location associated with that specific reciept. If the ICENH bin location for that nonstock product has not been loaded, it will be loaded from the POEI record as the non stock's default bin location (on the ICENH).
