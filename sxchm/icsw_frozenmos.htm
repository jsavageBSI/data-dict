icsw.frozenmos - Frozen Months
  
Label: Frozen Months
Type: integer
Format: >9
Decimals: ?
Initial: 6
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: Warning: Frozen Months Should Not be Changed (8632)
Trigger: 



Description:
Indicates the number of months frozen. This field can be set by either
the user or the computer (through ICAM). This field is used by ICAM to
determine when the product should be 'unfrozen' and brought back into
the ICAM calculation loop.
If set to '99' (the system will never set to this) the product will be
considered frozen forever.
