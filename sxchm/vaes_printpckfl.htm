vaes.printpckfl - Reprint Pick Ticket
  
Label: Reprint Pick Ticket
Type: logical
Format: yes/no
Decimals: ?
Initial: yes
Extent: 0
Mandatory: yes
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
This flag indicates whether the pick ticket needs to be printed or not.
When a value add transaction is initially added, this flag is set to yes
to indicate that a pick ticket does need to be printed. Once printed,
this flag is flipped to no. If a line is maintained or added, the
printpckfl will again be set to yes to allow a reprint of the pick
ticket.
