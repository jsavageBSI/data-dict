apei.reconty - Recon Type
  
Label: Recon Type
Type: character
Format: X
Decimals: ?
Initial: l
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: Reconciliation Method (L)ine, (B)ack Order, (I)tem
Trigger: 



Description:
Reconciliation Method Type:
L= match invoices to a specified PO line
B= match invoice to a specific PO line and any back orders for the line
I= match invoice to any line on the PO for that same item/product. The default method is "I"tem. However, the user will have the ability to override this method in the APEI center at the Invoice level.
