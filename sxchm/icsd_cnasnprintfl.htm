icsd.cnasnprintfl - Print Carton Detail on ASN
  
Label: Print Carton Detail on ASN
Type: logical
Format: yes/no
Decimals: ?
Initial: no
Extent: 0
Mandatory: yes
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
This option is controlled per warehouse only that is a Cartonization Whse.
If the option is no, then on the outboud BOD for the ASN - AdvanceShipNotice,
the carton information will not be sent - header (Carton) or detail (inside Carton)
data.
If the option is set to yes, then the outbound BOD will send the Header and
Detail information that is attached to the order.
It uses the same Node structure that was put in place with the TWL
project and sending carton informaiton outbound. The difference is the
data sent is not based on OEEHP records, but off ICECNHDR and
ICSCNDTL record information on the order.
