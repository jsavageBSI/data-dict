sasc.icnsdowriteamt - NS/DO Write-Off Amt
  
Label: NS/DO Write-Off Amt
Type: decimal
Format: >>>>>>9.99
Decimals: 2
Initial: 0
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: Write-Off Amount for NS/DO sets
Trigger: 



Description:
Amount to automatically write-off when a transaction takes a NS/DO set quantity to zero and the amount is non-zero. Typically will be only a few cents due to rounding.
If the remaining amount is within the specified write-off amount GL postings and IC transactions will be generated automatically and the set will be closed as if the amount was written off in ICEAN.
If the amount is left zero than no automatic write-offs will occur.
