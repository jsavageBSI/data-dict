icetc.icswcost - ICSW Cost
  
Label: ICSW Cost
Type: decimal
Format: >>>>>>9.99999-
Decimals: 5
Initial: 0
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
This represents the SM/GL as it existed BEFORE the application of the
current posting function (the one creating this record). If
appropriate, this includes the average add-on figure. This field is
very important as it represents the only way of figuring a cost for the
valuation of inventory for an 'as of' trial balance. The ICRT report
uses this figure from the first record posted AFTER the cutoff date to
calculate total value. It is critical that every ICET record create
record this figure prior to any recalculations.
If the product is a special price/cost item (see ICSP) then this cost is
in the special price/cost unit, not the stocking unit. This is called
the 'raw' cost.
