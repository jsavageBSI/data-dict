icsd.cnctnposctnno - Carton Number Position in Carton ID
  
Label: Carton Number Position in Carton ID
Type: integer
Format: 9
Decimals: ?
Initial: 0
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
Whse using Carton ID Formatting on the Whse, there are four sections
that can be used to format the Carton ID.
One is the Carton Number. It is a sequentially system generated number that
is always used in the Carton ID, but the formatting allows selection of where
the number displays in the sections used.
If all four sections in use, then can be 1-4, three sections 1-3, two sections
1-2, or just the number - 1.
 
Content:
Position of the Carton Number in the Carton ID
