Description:
Override Reason Type - This field is updated by the ICSWU screen. It
reflects the reason that a user entered an Override Usage amount. It
must be a valid "Usage/Ord Qty Override" code as defined in SASTT.
The field is a table of 25 representing each month from the current
month thru 25 months ago (in reverse order). overreasty[1] is the
current month, overreasty[2] is last month, etc.
