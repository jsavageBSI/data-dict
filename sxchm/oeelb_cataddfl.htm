oeelb.cataddfl - Added From Catalog
  
Label: Added From Catalog
Type: logical
Format: yes/no
Decimals: ?
Initial: no
Extent: 0
Mandatory: yes
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
Indicates if an order as needed product was added from the catalog when
this line was added. If the product was not set up when the line was
added but was defined in the catalog file, the operator has the option
to create an ICSP/ICSW record automatically. If the new records are
created, this flag is set to yes. If the catalog product is pulled
into the order as a non-stock product, this flag remains no. OERE
reports all products added from the catalog file as lines are added.
 
Content:
yes - an ICSP/ICSW was created from a catalog record as this line was
added.
no - an ICSP/ICSW was not created from the catalog when this line item
was added.
