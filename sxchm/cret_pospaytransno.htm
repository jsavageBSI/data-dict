cret.pospaytransno - Positive Pay Transaction
  
Label: Positive Pay Transaction
Type: character
Format: x(30)
Decimals: ?
Initial: 
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: Positive Pay Transaction Number
Trigger: 



Description:
If the check being printed is drawn on a bank with a Positive Pay program
and CenPOS is the method used to send check information to the bank,
this field will contain the 10 digiit transaction number returned by CenPOS.
If this check is ever voided, the transaction number can be sent back to
CenPOS for it to use to find information about the original transaction.
 
Content:
10 digit transaction number
