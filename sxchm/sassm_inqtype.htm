sassm.inqtype - Inquiry Type
  
Label: Inquiry Type
Type: character
Format: x(1)
Decimals: ?
Initial: 
Extent: 7
Mandatory: no
Val Exp: ?
Val Msg: 
Help: (J)ump, (P)ulldown or (T)rend Reserved
Trigger: 



Description:
Contains a type code to determine how the function key logic will be
processed. This field is updated using sassmi.p.
 
Content:
j = jump box logic. This will cause Trend to exit the calling program
and use the jump logic to validate security and call the program. A
SASSM record must exist for this calling program.
p = Pull down menu. This will find the SASSM record using the
sassm.inqproc field and display an updated function key bar.
t - Trend reserved. This will skip any SASSM requirements and simply
call directly the sassm.inqproc program. When the program has finished
executing control will return back to the calling program.
