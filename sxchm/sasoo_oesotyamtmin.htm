sasoo.oesotyamtmin - Min Hold Dollar Value
  
Label: Min Hold Dollar Value
Type: decimal
Format: zzzzzzzz9.99
Decimals: 2
Initial: 0
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: Minimum Dollar Value For SO or BR Type OE Order
Trigger: 



Description:
This field contains the minimum value for an SO or BR type Order Entry order. When this operator enters an SO or BR type OE order, the total ordered amount of that OE order will be compared against the value in this field. If the total ordered value of the OE order is less than the value in this field, the OE order will be placed on hold.
 
Content:
This field contains a positive numeric value that represents the minimum total ordered value for which this operator can enter an SO or BR type Order Entry order without that OE order being placed on hold.
A value of zero indicates that there is no minimum amount for this operator.
