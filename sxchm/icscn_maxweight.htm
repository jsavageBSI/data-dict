icscn.maxweight - Maximum Weight
  
Label: Maximum Weight
Type: decimal
Format: zzzzzz9.99
Decimals: 2
Initial: 0
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
Items total weight cannot exceed this max weight for the package
The Maximum Weight value cannot be less than or equal to the
Minimum Weight if the Minimum Weight setting is loaded (not Zero).
If the Maximum Weight on the packages is Zero, then there is assumed
to not be a weight limit on the package.
The weight is always measured in pounds.
