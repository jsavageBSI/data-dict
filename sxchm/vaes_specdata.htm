Description:
All value add transaction sections can contan specification/instruction
information. This is a free-form text area containing 11 lines of
60 characters on the screen, but an unlimited number of lines can be
entered. The specdata field contains the data which is the result of
a Progress V7 editor widget screen.
 
Content:
Specification/instruction free-form text data.
