icsc.cnpkggrouplist - Packing Group Type List
  
Label: Packing Group Type List
Type: character
Format: x(100)
Decimals: ?
Initial: 
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help:

Description:
Packing Restriction field (icsp.cnpkgrestrictty) must be 'Group Type Only'
to allow selection of a Type
Allows for No Selection, One Selection or Multiple Selections
The auto cartonization function will match the types based on the 'most'
items with the same types. Not always all types, but matching types to
other items and packages.
 
Content:
SASTT - Package Group Type
Stored as a comma delimited list of types when using multiple types
