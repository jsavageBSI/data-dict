wteh.createdby - Created By
  
Label: Created By
Type: character
Format: x(8)
Decimals: ?
Initial: 
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg:

Description:
the operator initials of the person that created the WT. Also columns
5-7 indicate the function that created the WT
 
Content:
Values for cols 5-7 :
-WT = created in WTET
-IN = created in OEEPI
-OE = created in OEET
-VA = created in VAET
-RR = created in WTERR
-PO = created in POERR
-CP = created in WT backorder process
