icsw.baseprice - Base Price
  
Label: Base Price
Type: decimal
Format: zzzzzz9.99999
Decimals: 5
Initial: 0
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
The base price is a customer price for this product in this location.
It is commonly used as a multiplier for PDSC records. As such, it can
represent a particular column price, the lowest acceptable price, or the
published price. This field can be updated manually or by a number of
PD functions, inlcuding automatic update tapes. Base price is used by
all pricing programs, including order entry.
Base Price is always in stocking units unless the product has special
price/cost units, in which case it is in the special price unit.
