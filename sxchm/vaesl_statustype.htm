vaesl.statustype - Status
  
Label: Status
Type: logical
Format: active/inactive
Decimals: ?
Initial: yes
Extent: 0
Mandatory: yes
Val Exp: ?
Val Msg: 
Help: (A)ctive, (I)nactive
Trigger: 



Description:
The status type identifies the status of the line item. It is used to
indicate if the line is still active, or has been cancelled or completed
 
Content:
active (true) - active line item
inactive (false) - cancelled or completed line item
