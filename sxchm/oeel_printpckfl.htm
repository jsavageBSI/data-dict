oeel.printpckfl - Pick Tck Print
  
Label: Pick Tck Print
Type: logical
Format: yes/no
Decimals: ?
Initial: yes
Extent: 0
Mandatory: yes
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
The print pick flag indicates if the pick ticket needs to be
printed/reprinted for this line item. It defaults to yes for each new
line added (indicating that a pick ticket needs to be printed for this
line). When the pick ticket is printed, it is set to no. If any changes
are made to the line item after picking, this field is reset to yes.
This field in conjunction with the prevqtyshp control the "Print Changes
Only" option on OEEPP.
 
Content:
yes = the pick ticket needs to be printed
no = the pick ticket has already been printed and no subsequent changes
have been made to the line item
