Description:
The origin code indicates where the OE order was created. The code may
be entered by the user or it may be loaded on the order automatically. The
areas where the code will be automatically are:
SX.api
PCM Configurator
Consolidated Invoice
Tool Crib
Counter Sale
EDI Import
Excel Import
Job Management
Receive On Account
eStorefront
Storerooom
Service Warranty
Other
 
Content:
A four character code indicating where the OE order was created.
The origin codes are set up in SASTT.
