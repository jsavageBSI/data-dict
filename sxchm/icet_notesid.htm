icet.notesid - Notes ID
  
Label: Notes ID
Type: integer
Format: zzzzzzzzz9
Decimals: ?
Initial: 0
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: Unique ID for Notes Creation to a Specific ICET Record
Trigger: 



Description:
This is an incremetal number that is generated from a 'Sequence' in the ICET
record called icetnotesid. As a note is attached to a specific ICET record,
the sequence is created and used as the unique connection to the NOTES
record and the ICET record. Since ICET does not have a primary/unique
index, the sequence id is required to keep the link specific to the records.
 
Content:
numeric sequence
