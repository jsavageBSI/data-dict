Label: 852 Out
Type: date
Format: 99/99/99
Decimals: ?
Initial: ?
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: Updated During EDI 852 With Current Date
Trigger: 



Description:
This field will indicate that last time this product/warehouse was
reported on the EDI 852 document. If it blank, it has never been
reported. It will be updated at the end of the 852 processing, but is
output to the 852 flat file prior to the updating. This will allow the
Vendor to know that if the field is blank, this is the first time the
product has been reported.
While this field is enterable in ICSWB, it should not be manipulated
except in certain situations. Once such situation would be that the
customer is converting to TREND and was using the 852 on their prior
system, then they would want to update this date either with the
conversion process or manually to show that this product was previously
transmitted on an EDI 852.
