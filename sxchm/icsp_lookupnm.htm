icsp.lookupnm - Lookup Name
  
Label: Lookup Name
Type: character
Format: x(15)
Decimals: ?
Initial: 
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 

Indexes:
k-lkup


Description:
The lookupnm is used as a secondary index for the ICSP file (the primary
is product). Lookup Name can default from the product id, the first
description, or the second description (based on ICAO option). Careful
consideration should be given as to the lookup name value. Examples:
Good:
Bit, 1/4" C
Bit, 1/2" C
Bit, 1/4" M
Bit, 1/2" M
Bad:
1/4" C Bit
C Bit, 1/2"
Bit, M 1/2"
