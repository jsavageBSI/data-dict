sasoo.cfgaccesscd - Configurator Access Code
  
Label: Configurator Access Code
Type: character
Format: x(1)
Decimals: ?
Initial: N
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: (V)alue Add, (N)o Access
Trigger: 



Description:
This field indicates whether or not the operator can access the Configurator
from Value Add.
Because VAET can be accessed when working with an OE order tied to a
VA order, this option can be set to prevent the OE operator from having
access to the configurator from the VA order.
 
Content:
Blank - User cannot access the Configurator from VA
V - User can access the Configurator from VA
