Description:
The name of the database table containing the field specified in the
FIELDNM field which can hold the name of an individual. When a GDPR
search is performed from the SAAG function, Elasticsearch will look in the
field on this table to see if the name matches the search value.
If the individulal is placed on a GDPR restriction, this field on this table will be
used to indicate to the UI that it may need to be hidden to prevent the
propagation of the name.
