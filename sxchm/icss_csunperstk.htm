icss.csunperstk - Price/Cost Units Per Stk Unit
  
Label: Price/Cost Units Per Stk Unit
Type: decimal
Format: zzzzz9.999999
Decimals: 8
Initial: 1
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
Represents the special pricing/cost unit conversion. In particular, the
number of pricing units per stocking unit. This field is used only if
using the special pricing/costing unit conversion (see speccostty).
Note that this field cannot be negative, but will often be less than
one. Note also the direction of conversion - number of special units in
a stocking unit. Examples:
Item Priced by Hundred -
csunperstk = .01 unitstock = "EACH"
Item Priced by Board Foot -
csunperstk = 3.5 unitstock = "EACH"
(meaning this board is equal to 3.5 boardfoot)
This conversion factor will be used for all pricing. In the first
example, a price of $10 would equal .10/EACH (10 * .01 * Qty). In the
second example, a price of $10 would equal $35/EACH (10 * 3.5 * Qty).
