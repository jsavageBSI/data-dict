sasgt.countryoforigin - Country of Origin
  
Label: Country of Origin
Type: character
Format: x(2)
Decimals: ?
Initial: 
Extent: 0
Mandatory: yes
Val Exp: ?
Val Msg: 
Help: 
Trigger: 

Indexes:
k-sasgt


Description:
The country associated with the tariff.
If a country is loaded on the tariff record, it will only be applied to products
where the POEL.COUNTRYOFORIGIN matches the country on the SASGT
record.
If no country is loaded on the tariff record, it will be applied to products where
the POEL.COUNTRYOFORIGIN is not loaded or no tariff code can be found
for the country that was loaded.
 
Content:
The 2 character abbreviation for a country as set up in the valid country
codes defined in SA Table Code Values Setup (SASTT).
