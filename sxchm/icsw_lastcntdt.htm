Description:
This field indicates that last time this product was counted using
either physical or cycle counting. It is updated automatically through
ICEPP and ICEPC. If the count is cancelled it is reset.
It is possible to run physical counts by picking up only those products
not counted since a certain date. This field is used for that report
criteria.
