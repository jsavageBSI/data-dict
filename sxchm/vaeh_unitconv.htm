vaeh.unitconv - # Stk Units
  
Label: # Stk Units
Type: decimal
Format: >>>>>9.99999
Decimals: 5
Initial: 0
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
The unit conversion factor indicates the number of stocking units in a
selling unit. If there are 24 stocking units in a case, the unitconv
would be = 24. If we are selling 1/6 of a stocking unit, the unit conv
would be .16667. This field is used to set the stkqtyord and stkqtyship
in the value add transaction for the finished product.
 
Content:
The factor as identified in ICSEU or SASTA.
