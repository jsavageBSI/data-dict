oeel.returnty - Return Type
  
Label: Return Type
Type: character
Format: x(1)
Decimals: ?
Initial: 
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: (V)endor, (S)tock, (U)navailable, or (P)artial Unavail
Trigger: 



Description:
The return type is used on a return line to indicate the disposition of
the returned product. The return type can represent a vendor return,
return of stock to unavailable (if damaged...), a return into stock, or
a partial unavailable return (some of the return will go into stock and
some will be set to unavailable). If a vendor return is selected, OEROW
can be used to assist in identifying items that still need to be placed
on a PO-RM.
 
Content:
v = vendor return
u = return to unavailable stock
s = stock return
p = partial unavailable/stock return
