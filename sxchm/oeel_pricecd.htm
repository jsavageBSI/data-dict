oeel.pricecd - Price Level/C+ %
  
Label: Price Level/C+ %
Type: decimal
Format: >>9.9999
Decimals: 4
Initial: 0
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
The price code stores additional price calculation information.
 
Content:
If cost plus pricing is used, this field stores the Cost Plus percent.
If margin pricing is used, this field stores the Margin percent.
if list pricing is used, this field stores the multiplier off of list
if base pricing is used, this field stored the multipler off of base
If Level pricing is used, this field stores the level selected.
All of these options are available in line item entry via the F12 key.
oeel.priceclty determines the pricing that was used.
