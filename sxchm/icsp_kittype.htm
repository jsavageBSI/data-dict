icsp.kittype - Kit Type
  
Label: Kit Type
Type: character
Format: x(1)
Decimals: ?
Initial: 
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: (P)rebuilt, (B)uild on Demand or Blank if not a Kit
Trigger: 

Indexes:
k-kit


Description:
Type of kit, if any. Normally blank, this field is used only for kit
products. Prebuilt kits are built using work orders in KPET. A
prebuilt kit is expected to be found on the shelf, and the quantity on
hand will normally by above zero. Usage is tracked on a prebuilt kit.
Build on demand kits are rarely true products on the shelf. Instead,
they serve as a 'shorthand' for a collection of products to be gathered
and, if appropriate, assembled at shipping time. Availability of build
on demand kits is based on the availability of the components.
 
Content:
B-Build on Demand None on the shelf, built at shipping time
P-Prebuilt Qty on the shelf, restocked from KPET
BLANK Not a Kit Item
M-Mix used for tally/mix item
