sastp.purlevelno - Purchase Level
  
Label: Purchase Level
Type: integer
Format: zz9
Decimals: ?
Initial: 0
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
Purchase Card Level Number. This is used in the ICVerify Credit Card interface to determine what level of data to send to the ICVerify interface during the settlement process. When the level is set to level 2 or 3, the processor being used has to be configured to correctly handle the data being send.
 
Content:
0 - default (same as level 1)
1 - Level 1 normal processing
2 - Level 2 Purchase card
3 - Level 3 Prurchase Card level data (line level)
