Description:
The original promise date of a given order line. Defaults in from the promise
date and can only be changed when the origpromisedtlockedfl is false and
the user has security (sasoo.origpromisedtfl) and the sasoo setting for
tracking promisedt at the line level is true.
 
Content:
Promisedt prior to any changes made to that value or ? if not carried at
the line level.
