icsp.cutparttype2 - 2nd Part Type
  
Label: 2nd Part Type
Type: logical
Format: Decimal/Integer
Decimals: ?
Initial: yes
Extent: 0
Mandatory: yes
Val Exp: ?
Val Msg: 
Help: Enter the 2nd Part Entry Type - (D)ecimal or (I)nteger
Trigger: 



Description:
SXRF Cut Product Part Type #2
If the product is set up as a Linear Cut product (icsp.cuttype = L), the
length of a cut piece can be entered as one number (feet) or as two
numbers (feet and inches). This field is used to indicate how the second field
should be formatted.
The length value can be entered as a whole number (Integer) or it can be
entered as a decimal number (Decimal)
 
Content:
Decimal= yes
Integer= no
