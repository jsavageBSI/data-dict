icselc.width1 - Width 1
  
Label: Width 1
Type: decimal
Format: zzzzz9.99999
Decimals: 5
Initial: 0
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
Used to store one piece of the dimensions for a specific piece of a
dimensional product.
A dimensional product can either be a rectangle or a step cut. With a
rectangle, the user will enter 2 lengths and 2 widths. With a step cut, the
user will enter 3 lengths and 3 widths. The dimensions are entered starting
from the upper left corner and working counter-clockwise around the piece.
The first width value is the left-most edge of the piece. Only 90 degree
angle cuts are allowed with the dimensional functionality.
