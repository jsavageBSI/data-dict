gdprctrl.sourcecd - Data Source
  
Label: Data Source
Type: character
Format: x(2)
Decimals: ?
Initial: 
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
A two character code used to group fields containing data about an
individual into functional areas. The user can select one or more of these
areas when searching for instances of an individual in the application.
 
Content:
AP - (Accounts Payable) APSF, APSS and APSV
AR - (Accounts Receivable) ARSC, ARSD, ARSG and ARSS
CN - (Contacts) CONTACTS
EM - (Employees)PV_USER, SASTA and SMSN
PR - (Prospects) CMSP
WL - (Warehouse Logistics) CARRIER, EMPMST, VENDADDR and VENMST
