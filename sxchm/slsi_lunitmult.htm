slsi.lunitmult - Stnd Pack Multiplier
  
Label: Stnd Pack Multiplier
Type: integer
Format: z9
Decimals: ?
Initial: 0
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: Length of Standard Pack Multiplier
Trigger: 



Description:
This field indicates the length of the standard pack multiplier in the
Supplier Link data file. The Supplier Link import will read in the
data beginning at the starting position going for the number of
characters specified by the length value.
For example, if the standard pack multipler starts in position 34 and has a length of 10, the import program will read in whatever is in positions 34, 35, 36, 37, 38, 39, 40, 41, 42 and 43 as the standard pack unit and will store it as a decimal value.
 
Content:
An integer value indicating the length of the data in the Supplier Link data file that contains the value of the standard pack multiplier.
