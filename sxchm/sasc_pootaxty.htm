sasc.pootaxty - POO Tax Type
  
Label: POO Tax Type
Type: character
Format: x(1)
Decimals: ?
Initial: F
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: Load POO With Ship (T)o,Ship (F)rom,(C)all Center or (B)lanks
Trigger: 



Description:
This field will store how to load the Point of Order Origin (POO) address information that will be passed to Taxware.
There will be four possible values:
F - Ship From - the ship from address taken from the ICSD warehouse record where the order was placed
T - Ship To - the ship to address taken from the OE order header
C - Call Center - the address taken from the SMSN operator record for the 'Taken By' operator on the OE order
B - Blanks - blanks will be passed for all address data; it is assumed that the customer has set up NexPro data for all applicable taxing nexus locations.
The default will be F - Ship From - which is how the POO address has always been loaded in standard SX.enterprise.
