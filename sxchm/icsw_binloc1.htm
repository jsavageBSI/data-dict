icsw.binloc1 - Bin Loc #1
  
Label: Bin Loc #1
Type: character
Format: xx/xx/xxx/xxx
Decimals: ?
Initial: New Part
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: Building / Row / Section / Shelf
Trigger: 

Indexes:
k-bin


Description:
First bin location for this product. Bin locations are in the format
BB/RR/SSS/FFF B-Building R-Row S-Section F-Shelf
Bin locations are not required but are heavily used by all physical and
cycle count operations. Implementing physical counts without bin
locations will be difficult. Implementing cycle counts without bin
locations is impossible.
Every product should be assigned to a unique bin location for a given
warehouse. The system does not ensure uniqueness. Bin location 1 is
considered the primary location where bin location 2, if present, is the
overflow or secondary location.
If the product is warehouse manager controlled the bin locations in ICSW
will not be used.
