icets.custno - Customer #
  
Label: Customer #
Type: decimal
Format: zzzzzzzzzzz9
Decimals: 0
Initial: 0
Extent: 0
Mandatory: yes
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
This field is used when the ordertype field is "p" or "o".
If the ordertype field is a "p", then the vendor number of the purchase
order is stored in this field.
if the ordertype field is a "o". then the customer number of the sales
order is stored in this field.
