Description:
The warehouse on the value add transaction line item. A transaction can
be entered for only a single warehouse. (If inventory is to be relieved
in a different warehouse, a WT order can be used.) The warehouse on the
value add transaction is the warehouse the transaction is picked from
and the warehouse that gets credit for the sale. The warehouse on the
line item is the internal processing whse for an "IT" (Internal Process)
section and the primary warehouse from the banner for all other
sections.
 
Content:
Validated against ICSD - must be a warehouse that can be sold from.
