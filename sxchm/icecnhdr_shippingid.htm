k-shippingid


Description:
The operator can select to manually enter a Shipping ID to be attached
to the Cartons via ICECN - when running Auto Cartonization.
The operator can enter in an ASN Number, a Truck Number. The ID is
a way to link all the orders in the Cartonization process together.
An easy way in ICECN to see all the cartons for a Shipment - Advance Search.
If a Shipping ID is chosen to be created automatically (from ICECN or the
Picking runs - OEEPP/WTEP), it will be in the following format:
YYYYMMDD-HH:MM:SS-zz9
