arsc.cncartonty - Use Cartonization
  
Label: Use Cartonization
Type: character
Format: x(2)
Decimals: ?
Initial: 
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
Use Cartonization or not. Only visible if the AO setting is turned
to use cartonization system wide. If the value is blank, the customer use
of cartonization defaults to the Warehouse Use Cartonization setting. If
the setting is NC, then the customer will never use cartonization
 
Content:
Warehouse Default (Blank)
Auto Assign Order Entry Orders (AO)
Auto Assign Warehouse Transfers (AT)
Auto Assign Orders and Transfers (AB)
Manually Assign Order Entry Orders (MO)
Manually Assign Warehouse Transfers (MT)
Manually Assign Orders and Transfers (MB)
No Cartonization (NC)
