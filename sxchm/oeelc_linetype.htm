Description:
This field indicates the stand alone comment line type. "I"nternal
line types will display on internal screens and documents only.
"e"xternal line types will display on all internal and external
documents. Sub"t"otal line types will carry the subtotal description
as well as an indicator as to where the subtotal will be displayed
after.
