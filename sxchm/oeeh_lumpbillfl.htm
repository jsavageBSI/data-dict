oeeh.lumpbillfl - Lump Sum Bill
  
Label: Lump Sum Bill
Type: logical
Format: yes/no
Decimals: ?
Initial: no
Extent: 0
Mandatory: yes
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
Indicates if the lump billing amount should be used when invoicing the
customer. The flag allows the flexibility to invoice the customer a 0
amount on a particular invoice.
 
Content:
yes = this is a lump bill order
no = this is a normally billing order
