Description:
Is the order an original order fulfillment order. The order that all the items
were placed on before stock orders are created from the warehouses
selected to fulfill the order. It will always be a Blanket Order (BL). This
allows for no affect on the inventory in the system. This order will also
be used in Consolidated Invoicing for billing of one invoice based on
all the Tied Fulfillment Orders created from the original Fulfillment Order
