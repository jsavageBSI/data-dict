sasoo.oedotymrgpctmin - DO Min Hold Margin %
  
Label: DO Min Hold Margin %
Type: decimal
Format: zzz9.99
Decimals: 2
Initial: 0
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: Minimum Margin Percentage For DO Type OE Order
Trigger: 



Description:
This field contains the minimum margin percentage for a DO type Order Entry order. When this operator enters a DO type OE order, the margin percentage based on the ordered amount for that OE order will be compared against the value in this field. If the margin percentage on the OE order is less than the value in this field, the OE order will be placed on hold.
 
Content:
This field contains a positive numeric value that represents the minimum margin percentage based on the ordered amount for which this operator can enter a DO type Order Entry order without that OE order being placed on hold.
A value of zero indicates that there is no minimum percentage for this operator.
