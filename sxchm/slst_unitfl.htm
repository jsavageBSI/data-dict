slst.unitfl - Unit Conversion Type
  
Label: Unit Conversion Type
Type: logical
Format: multiple/fraction
Decimals: ?
Initial: multiple
Extent: 0
Mandatory: yes
Val Exp: ?
Val Msg: 
Help: M)ultiple or F)raction of Stocking Unit
Trigger: 



Description:
This field is used as a default when a new ICSEU or SASTT Type U unit record is created for a standard pack unit that does not already exist.
If the field is multiple then the unit conversion is the value imported from the Supplier Link data file. For example, if a value of 10 was imported, the conversion factor for the new unit will be 10.
If the field is factor then the unit conversion is 1 divided by the value imported from the Supplier Link data file. For example, if a value of 10 was imported, the conversion factor for the new unit will be 0.1 or 1/10.
 
Content:
A logical field that will indicate whether the unit conversion should be a multiple (1 times the standard pack multiplier) or a fraction (1 divided by standard pack multiplier).
