Description:
This is a specific counter based on the type (territorycd). We do not carry all
the 'extra' fields based on type of record - custno, shipto, salesterr, custtype,
pricetype, etc in the primary unique index, but use the counter based on the
type:
cono
srcrowpointer (the parent ICSPR record)
territorycd
seqno
Will always fidn the 'last' seqno based on type and add one to it.
