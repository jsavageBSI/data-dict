icss.icspecrecno - Spec Rec Link
  
Label: Spec Rec Link
Type: integer
Format: >>>>>>9
Decimals: ?
Initial: 0
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: Special Price/Cost Record Number for Link to Transactions
Trigger: 

Indexes:
k-icss k-status


Description:
This field will be incremented by one for each cono/prod combination.
It is the third portion of the primary unique key, with cono and prod.
It will be used to identify the special price/costing record to the
transaction it is related to. Each time the special price/costing data
is changed a new icss record will be created and this number will be
based on the the last record for the cono/prod and incremented by one.
This field will begin with zero for each cono/prod.
