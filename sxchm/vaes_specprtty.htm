vaes.specprtty - Specifications Print Type
  
Label: Specifications Print Type
Type: character
Format: x(1)
Decimals: ?
Initial: 
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
This flag is used with the specification/instruction free-form text
information (specdata) and indicates how this data should be printed.
The specprtty field is used for the "SP" (specification/instruction)
section itself.
 
Content:
The specprtty field indicates how to print the data:
E - External processes only
I - Internal processes only
B - Both External and Internal Processes
N - Neither
