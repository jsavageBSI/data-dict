arsg.selltype - Sales Order Status
  
Label: Sales Order Status
Type: character
Format: x
Decimals: ?
Initial: y
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: (Y)es (N)o (C)ash only CO(D) only (H)old until (O)pen until
Trigger: 



Description:
Determines the approval status of orders entered in OEET by a member of
the group.
If the group sell type is set to Y - YES, all orders entered by members of the group
will be approved if within all other credit parameters
If the group sell type is set to N - NO, members of the group will not be allowed to
enter orders in OEET.
If the group sell type is set to C - CASH ONLY, The only payment type accepted
for any member of the group will be cash.
If the group sell type is set to D - COD ONLY, the only orders the members of
the group can place willb e Cash On Delivery.
If the group sell type is set to H - HOLD UNTIL, all orders placed by members
of the group will be placed on hold until the group Hold Date is reached.
Once the group's hold date has been reached, all new orders will be approved.
If the group sell type is set to O - OPEN UNTIL, all orders placed by members
of the group will follow the normal credit process until the group Open Date is
reached. After that date, all new orders placed by members of the group will
be placed on hold.
If the group sell type is blank, the approval status check will bypass the group
to the individual members of the group. The Sales Order Status set up on the
customer record will be used.
 
Content:
Valid Values:
Y, N, C, D, H, O and blank
