icetc.usageprod - Usage Product
  
Label: Usage Product
Type: character
Format: x(24)
Decimals: ?
Initial: 
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 

Indexes:
k-usgwhse


Description:
This field will be blank if icet.usagefl is set to "no". If icet.usagefl
is set to "yes", usageprod will be the product whose usage is updated
during icam. In most cases the value of usageprod will be the same as
icet.prod. But if a substitute occurs, icet.prod will contain the
substituted product which was actually sold (or transferred) and
usageprod will contain the original product which was substituted.
During the icam run, the usage will be added to the original product
that is stored in this field.
