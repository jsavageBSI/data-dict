Description:
If a product has been defined as a regrind product, that product, when
consumed, is able to be reground, sharpened, repaired or refurbished into a
usable product. A labor product can be associated with a regrind product
which will be used as part of the regrind process.
 
Content:
A valid ICSP product that is set up as a labor product.
