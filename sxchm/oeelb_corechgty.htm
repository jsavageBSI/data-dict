oeelb.corechgty - Core Charge
  
Label: Core Charge
Type: character
Format: x
Decimals: ?
Initial: 
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: (Y)es, (N)o, or (R)eturn
Trigger: 



Description:
The core charge type. If the line is eligible for a core charge (ICSP
core charge is &gt; 0), the core charge will be enterable from the extended
screen. A core can also be returned. This field distinguishes between
the 3 situations.
 
Content:
y = this line is eligible for a core charge
n = this product does not have a core
r = this is a core return line
