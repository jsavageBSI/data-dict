Val Exp: ?
Val Msg: 
Help: Beginning Column or Field Number of Cost Breaks
Trigger: 



Description:
This field describes where in the Import file the Product Cost Breaks
can be found.
.
If the Import File Type (slsi.filetype) = "f" (Fixed Length) then this
number indicates the beginning column of the Cost Breaks (one for each
element of the array).
If the Import File Type (slsi.filetype) = "p" (Progress) or "d"
(Delimited) then this number indicates the field number of the Cost
Breaks (one for each element of the array).
.
It is enterable by the user through Supplier Link - Setup Import
Definitions - SLSIQ (SLSI F8-Qty Breaks)
.
The field is used to contruct the SLED.SLCOSTBRK array.
.
Last changed dww 02/11/94
 
Content:
Numeric: 0-9999
Array: 9 elements
A value of zero means that this field isn't mapped and the Cost Break
(SLED.SLCOSTBRK) for that array element will not be imported.
