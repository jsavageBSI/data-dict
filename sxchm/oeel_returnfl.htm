oeel.returnfl - Return
  
Label: Return
Type: logical
Format: -/
Decimals: ?
Initial: no
Extent: 0
Mandatory: yes
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
The return flag indicates that this line item is a return line item. It
can be yes or no on the following order types: SO, CS, CR. The return
flag is always set to yes on an RM and is always no on the other order
types. The return flag displays as a "-" if the line is a return line.
