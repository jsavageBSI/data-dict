Description:
Added with JIRA SXWEB-38274/SXWEB-39784. This field stores the OE Product Type Cost Override for special cost overrides for labor products. These settings will allow the user to override the cost of a labor product in OEET, based on the scenario.
 
Content:
Value Description
LD Labor Items - Drop Ship Type
LS Labor Items - Both Specials and Drop Ships
LB Labor Items - Both Specials and Drop Ships
LA Labor Items - Any Type
LN Labor Items – Do Not Allow Cost Override
Blank Default
