Label: ?
Type: integer
Format: zzz9
Decimals: ?
Initial: 0
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: Beginning Column or Field Number of Description 2
Trigger: 



Description:
This field describes where in the Import file the second Description can
be found for the Product.
If the Import File Type (slsi.filetype) = "f" (Fixed Length) then this
number indicates the beginning column of the second Description.
If the Import File Type (slsi.filetype) = "p" (Progress) or "d"
(Delimited) then this number indicates the field number of the second
Description.
It is enterable by the user through Supplier Link - Setup Import
Definitions - SLSIP (SLSI F7-Product).
It is used to construct SLED.SLDESCRIP[2].
Last Changed dww 02/11/94
 
Content:
Numeric: 0-9999
A value of zero means that the second Description is not mapped and will
not be imported.
