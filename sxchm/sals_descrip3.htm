Description:
This field holds the extended product description which can be used to
identify a product.
Individual words in the description are also loaded as a part of the keyword
system. To best utilize keywords, make sure to put spaces, not punctuation,
between the words of the description. For example:
GOOD: 1/2" Carbide Drill Bit, 3/8" Shank
BAD: 1/2"-Carbide Drillbit-3/8"Shank
 
Content:
A language specific interpretation of the extended product description
