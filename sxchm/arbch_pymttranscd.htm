arbch.pymttranscd - Trans Code
  
Label: Trans Code
Type: character
Format: x(2)
Decimals: ?
Initial: 
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
This field is used to describe the type of transaction. If the check number is less than 8 digits, the field is loaded with "CK". If the check number is greater than 8 digits (a number, an alpha check number, or a credit card), the field is loaded with "CK" for check number,"AC" for alpha check number or a user defined value such as "CC" for credit card.
