apsv.esb855isetnotesprintfl - Set Print Status for Notes
  
Label: Set Print Status for Notes
Type: logical
Format: yes/no
Decimals: ?
Initial: no
Extent: 0
Mandatory: yes
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
Used only in the 855in Purchase Order Acknowledgement ION/BOD
interface.
It is used as a replacement of option 3 of the EDEPA 855in report.
If yes, any incoming notes or comments will be added with a print status of
yes.
If no (default), any incoming notes or comments will be added with a print
status of no.
