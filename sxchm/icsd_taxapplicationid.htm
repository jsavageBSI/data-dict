icsd.taxapplicationid - Tax Organization Name
  
Label: Tax Organization Name
Type: character
Format: x(40)
Decimals: ?
Initial: 
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
For AvaTax the Tax Organization ID is normally stored by company (in AO - Integrations - Tax Interface Options) nd is used when calling AvaTax.
If Tax Organization by Warehouse is enabled in SASC - Taxing then the Tax Organization ID can be specified by Warehouse.
If this field is loaded in ICSD it will be used with AvaTax, otherwise the AO Tax Organization ID will be used for the AvaTax call.
