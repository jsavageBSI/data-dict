Description:
The order number of the order. This is the primary key.
When a note is entered in Storeroom, the API contains 6 fields employee #,
name, department, work order, project and machine #. A note is
created which is visible in SX.e. In addition, the oeehextra table is
populated with the field values. The oeehextra table is tied to the OE order.
