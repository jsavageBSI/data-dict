icsp.cutnumparts - # Parts
  
Label: # Parts
Type: integer
Format: 9
Decimals: ?
Initial: 0
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: Number of Entry Fields for Cut Allocation Entry, Enter 0 When Not a Cut Part
Trigger: 



Description:
SXRF Cut Product Number of Parts
If the product is set up as a Linear Cut product (icsp.cuttype = L), the
length of a cut piece can be entered as one number (feet) or as two
numbers (feet and inches). This field indicates how many fields should be
displayed to the user when the user is enteing length data.
 
Content:
1 - Display one field for the user to enter the length of the cut piece
2 - Display two fields for the user to enter the length of the cut piece
