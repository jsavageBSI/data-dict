oefill.processty - Process Type
  
Label: Process Type
Type: character
Format: x(1)
Decimals: ?
Initial: 
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 

Indexes:
k-oefill k-order k-prod


Description:
The process type is used to group fill records according to their type
so similar records can be processed/reported together.
 
Content:
blank = a back ordered WT/OE/component line to be filled
p = a back ordered WT/OE/component line that has already been
filled (used to re-read records to print them on the report)
s = special/non-stock/tie records that were previously filled
(used to re-read the records to print them on the report)
