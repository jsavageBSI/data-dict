icsw.addoncost - Addon
  
Label: Addon
Type: decimal
Format: zzzzzz9.99999-
Decimals: 5
Initial: 0
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
Average addons. This field is used whenever addons are capitalized from PO and/or WT. The figure represents the average of all addons for all items in stock. It is protected by the 'see cost' flag in SASOO. This figure is maintained by the system and should not be altered by the
user. The average addons figure is added to all outgoing costs if the
'Include Addons' question is set to true.
