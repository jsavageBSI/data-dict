icsw.nodaysso - # Days Stockout
  
Label: # Days Stockout
Type: integer
Format: >9
Decimals: ?
Initial: 0
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: Warning: Stock Out Information Should Not be Changed (8633)
Trigger: 



Description:
In days, the length of the last stock out for this product. This
information is maintained by the system in conjunction with the last
stock out date and the stock available date. These are set by the
various functions that relieve and receive goods.
