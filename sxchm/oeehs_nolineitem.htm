oeehs.nolineitem - # of Lines
  
Label: # of Lines
Type: integer
Format: >>9
Decimals: ?
Initial: 0
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
The number of lines items the match the price type for the found PDSS
record. This number is used in OEEPI for proration of the special
discount. Each line will get a percentage of the calculated discount
until the last line - it will get the remainder.
 
Content:
An accumulated number of line items for the price type.
