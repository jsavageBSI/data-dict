slsi.bcost - Cost
  
Label: Cost
Type: integer
Format: zzz9
Decimals: ?
Initial: 0
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: Beginning Column or Field Number of Cost
Trigger: 



Description:
This field describes where in the Import file the Product Cost can be
found.
.
If the Import File Type (slsi.filetype) = "f" (Fixed Length) then this
number indicates the beginning column of the Product Cost.
If the Import File Type (slsi.filetype) = "p" (Progress) or "d"
(delimited) then this number indicates the field number of the Product
Cost.
.
It is enterable by the user in Supplier Link - Setup Import
Definitions - SLSIP (SLSI F7-Products).
.
This field is used to construct the field SLED.SLCOST.
.
Last Changed: dww 02/11/94
 
Content:
Numeric: 0-9999
A value of zero means that this field isn't mapped and Product Cost
(SLED.SLCOST) will not be imported.
