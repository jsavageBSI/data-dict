pv_oeln.inventoryty - Inventory Type
  
Label: Inventory Type
Type: character
Format: x(1)
Decimals: ?
Initial: 
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
The inventory type will be used to indicate if the OE line was entered for
customer-owned inventory or distributor-owned inventory. This field will only
be loaded with a C or D if the order was placed in a storeroom mananged
warehouse. OE lines for customer owned inventory must be handled
differently because it is product that the customer has already purchased.
 
Content:
C - Customer owned inventory
D - Distributor owned inventory
blank - Distributor owned inventory
