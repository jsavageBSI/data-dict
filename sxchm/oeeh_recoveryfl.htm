oeeh.recoveryfl - Recovery Order
  
Label: Recovery Order
Type: logical
Format: yes/no
Decimals: ?
Initial: no
Extent: 0
Mandatory: yes
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
This field allows an order to be identified as a Recovery order. It will be
automatically set to yes for orders created from the ICRS report. This field
will not be available for a user to modify or to manually mark an OE order as
a recovery order.
A recovery order is an order created to bill the customer for inventory that has
not moved within a specified period of time. This is part of the Storeroom
functionality.
