icsp.cubes - Cube

Description:
The size, in cubes, represented by a single stocking unit of this
particular item. This field is generally for user information, printing
on pick tickets and a few reports. There are two situations, however,
where the value of this field becomes important:
1) Order Targets by Cube. Cube can be used as a target method in ICSL.
If so, the cube value * the quantity ordered is used for target and
minimum buy information. Note also that if target is cube, the
total volume in APSP will be by cube and review cycles will be
calculated using total cubes and target cubes.
2) If using warehouse manager and calculating bin capacity by cube,
this field is used to determine the total quantity of product that
can fit in a bin and the space available in any bin containing this
product. For this purpose it is important that cube is as accurate
as possible.
