icsd.cnctnidpkgtyfl - Use Package Type in Carton ID
  
Label: Use Package Type in Carton ID
Type: logical
Format: yes/no
Decimals: ?
Initial: no
Extent: 0
Mandatory: yes
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
Part of the 'Load Carton ID From Warehouse Settings' option.
If yes, then this field is available to control the use of the lpackage type in the
Carton ID or not.
The Package Type is loaded on each Package in the system. In SASTT,
there is a Package Type setup. It allows the used of the full Package Type
defined (ength up to 8) and connected to the Package, or the SASTT setup
can allow for up to three characters
Example: Full Type = Dry Ice
Carton ID = DI
 
Content:
yes - using the Package Type in the Carton ID format
no - not part of Carton ID's
