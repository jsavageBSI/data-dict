vaes.processinit - On Line Processing Initials
  
Label: On Line Processing Initials
Type: character
Format: x(4)
Decimals: ?
Initial: 
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 

Indexes:
k-process


Description:
This field indicates if an on line process (VAES, POEI, WTEI, VAEI) has
begun for this value add transaction section. The processinit field
contains the initials of the operator who started the on-line portion of
the process, but has not completed the processing (F10-Update). This
field works in conjuction with the vaes.processfl field.
 
Content:
operator initials doing the process
