icsl.class - Product Class
  
Label: Product Class
Type: integer
Format: >9
Decimals: ?
Initial: 1
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: Assigned by Product Classification Report (1 - 13)
Trigger: 



Description:
The class field can be maintained by the user or can be set by the
inventory classification system (ICAI). The system uses the Gordon
Graham 1 - 13 classification system as oppossed to the more traditional
A/B/C method.
 
Content:
1 - Top 7-1/2% of the items being classified
2 - Next 7-1/2 %
3 - Next 10%
4 - Next 10%
5 - Next 8% (repeat through class 11)
12 - Last 9%
13 - Dead Stock
Note that the percents are applied against the total number of SKU's
being classified. All products are sorted from highest dollars through
inventory to lowest dollars through inventory. The dead stock threshold
is drawn and all products below it are eliminated. The total number of
products is divided into the percentages above and cutoff lines are
determined. Classification is then done.
