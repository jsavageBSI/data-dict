Description:
The SASTAE table was designed to store extended data associated with data set up in the SASTT System Table. This field serves as a link between the SASTT setup data and the SASTAE extended data.
For example, you set up Warehouse Groups in SASTT using the code 'WG'. The SASTAE table is used to store the warehouses that make up a warehouse group and, therefore, have the code 'WG' as well.
 
Content:
This field contains a character value that represents one of the valid types in the SASTT function.
