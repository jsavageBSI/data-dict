vaesl.usagefl - Usage
  
Label: Usage
Type: logical
Format: yes/no
Decimals: ?
Initial: no
Extent: 0
Mandatory: yes
Val Exp: ?
Val Msg: 
Help: Record Usage
Trigger: 



Description:
The usage flag is used to determine if usage in ICSWU should be updated
for this line item. It is defaulted in the following manner:
no - a labor product
- a non-stock line
- a special order item that is not an order as needed product
- a stock product on a DO order
yes - all other combinations
The usage flag is overrideable on the line extended screen. (It is not
overrideable on a labor product, non-stock product, lost business line,
or core return). When the value add transaction section is completed, an
ICET record is created. The ICET record contains a usagefl and
mergedfl. ICAM uses ICET record to record used. If the ICET mergedfl =
no and ICET usagefl = yes, ICAM will use this ICET record to record
usage in ICSWU.
 
Content:
yes = usage will be recorded for this line
no = usage will not be recorded for this line
