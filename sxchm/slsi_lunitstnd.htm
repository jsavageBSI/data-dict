slsi.lunitstnd - Standard Pack
  
Label: Standard Pack
Type: integer
Format: 9
Decimals: ?
Initial: 0
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: Length of Standard Pack Unit
Trigger: 



Description:
This field indicates the length of the standard pack unit in the Supplier Link data file. The Supplier Link import will read in the data beginning at the starting position going for the number of characters specified by the length value.
For example, if the standard pack unit starts in position 34 and has a length of 4, the import program will read in whatever is in positions 34, 35, 36 and 37 as the standard pack unit.
 
Content:
An integer indicating the length of the data in the Supplier Link data file that contains the standard pack unit.
