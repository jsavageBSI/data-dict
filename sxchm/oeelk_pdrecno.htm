oeelk.pdrecno - PD Rec #
  
Label: PD Rec #
Type: integer
Format: >>>>>>>9
Decimals: ?
Initial: 0
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
The price/discounting record number (from PDSC) used to price this
component. If a PDSC record was not found for the component, the pdrecno
will be 0. If a PDSC record was used to price the line, the pdrecno is
loaded from that PDSC record. This field in conjunction with the company
number will allow direct access to the PDSC record. This direct access
is used to update the actual quantity on a month/yearly accumulated PDSC
record as well as to adjust for special costing (in p-oeeltm.i).
