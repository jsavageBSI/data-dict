vaes.specprtfl - Print Flag
  
Label: Print Flag
Type: logical
Format: yes/no
Decimals: ?
Initial: no
Extent: 0
Mandatory: yes
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
This flag is used with the specification/instruction free-form text
information (specdata) and indicates how this data should be printed.
The specprtfl field is used for all sections other than the "SP"
(specification/instruction) section itself.
 
Content:
The specprtfl field indicates how to print the data:
yes - print the data for this section
no - don't print the data for this section
