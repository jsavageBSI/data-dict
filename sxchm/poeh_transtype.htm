Description:
The transaction type of the order.
 
Content:
This describes the kind of order it is. Types include:
PO - Purchase order (Entry into stage 1)
RM - Return Merchandise
DO - direct order (Entry into stage 1)
QU - quote (Entry into stage 0. Converted to a PO using POEPC or POET File - Copy)
BL - blanket order (Entry into stage 0)
BR - blanket release is created from the BL
AC - accumulative
