icsd.wtrndonlyfl - WT Unit Round Only?
  
Label: WT Unit Round Only?
Type: logical
Format: yes/no
Decimals: ?
Initial: no
Extent: 0
Mandatory: yes
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
WTERR uses ICSW transfer unit for rounding when creating the WTERA Report record. This field will determine which unit to use on the WTERA Report.
Ex. Stocking unit = each, Transfer Unit = case (24)
System calculates 21 each are required and orders 2 case.
If WT Unit Round Only = yes then instead of 2 case, it will order 24 each. It will round, but will not use the WT Transfer Unit (Similar to POERR standard pack rounding)
