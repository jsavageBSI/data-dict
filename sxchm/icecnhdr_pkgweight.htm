icecnhdr.pkgweight - Package Weight
  
Label: Package Weight
Type: decimal
Format: zzzzzz9.99
Decimals: 2
Initial: 0
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
This is the combined weight of all the items in the package.
This is the weigh totalled from the ICSP Weight (Stocking)
converted to the unit conversion on the line of the order and then
multipled by the quantity in the box for the item.
A system generated weight. This will default in as the weight during shipping
of the package, but the opertor will enter in the Actual Weight of the
package to be compared to the Dimensional Weight.
The system weight is used during cartonization to find packages and compare
to package min/max weight settings.
