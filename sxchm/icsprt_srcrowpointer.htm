Description:
This is the Row Pointer of the Restriction record (ICSPR) which is the parent
record. There can be many specific ICSPRT records under one ICSPR record,
one record, or nor records. We use the row pointer since the parent record
could be prod, catelog, prod cat, vendno. prodline, brand code, alt prod
group, etc and no need to duplicate all those fields within the child table.
One to one match to the parent ICSPR record.
