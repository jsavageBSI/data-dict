Description:
If a cross-reference is found in ICSEC for type "c" (Customer's
Product) or "i" (Interchange), then this will be:
- the Customer Product (for type "c") directly from the flat file
- the Interchange Product (for type "i") directly from the flat file
Otherwise, this field will be blank.
