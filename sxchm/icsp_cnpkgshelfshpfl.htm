Description:
Yes - Excluded from Cartonization. Assumes in shipping package already
(box, case, pallet). No need to package the item. Ships in the container of
the unit on the order.
The Order line UOM must match the ICSP Stocking Unit of Measure to
be considered a Shelf to Ship item during Cartonization
If using auto cartonization, the system will assign a Carton ID per
package quantity on the order (1 of 5)
Assumes the Stocking Unit on the ICSP record is in a Carton not an Each
