icsw.nontaxtype - Non Tax Reason
  
Label: Non Tax Reason
Type: character
Format: x(2)
Decimals: ?
Initial: 
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
This field is used to store a type code for a non-taxable item,
indicating why the product is exempt from sales tax in this location.
Any entry is validated against SASTT type N. If the item is nontaxable
this is a required field. It is used by the tax analysis report (SMRT)
to group like items.
