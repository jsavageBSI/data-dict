oeeh.origcanfl - Original Canceled
  
Label: Original Canceled
Type: logical
Format: yes/no
Decimals: ?
Initial: no
Extent: 0
Mandatory: yes
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
This flag indicates that the original order was cancelled when the back
order was created. As a result of version 3.1 changes, this can rarely
happen anymore. The original is cancelled when the back order is created
with the exact makeup of the original order.
 
Content:
yes = the original order was cancelled when the back order was created.
no = the original order was not cancelled as a result of back order
creation.
