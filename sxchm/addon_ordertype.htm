addon.ordertype - Order Type
  
Label: Order Type
Type: character
Format: X(40)
Decimals: ?
Initial: 
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: Enter the Type of Document This Addon is Attached To
Trigger: 

Indexes:
k-addon k-seqno


Description:
Describes the type of addon being stored in the record. There are addons for orders, for POs, and for ARETT taxing records
 
Content:
OE = orderno and ordersuf are from OEEH
PO = orderno and ordersuf are from POEH (pono and posuf)
For ARETT addon records, the ordertype value is:
arett.recty + chr(3) + arett.taxgroup + chr(3) + arett.statecd + chr(3) +
arett.countycd + chr(3) + arett.citycd + chr(3) + arett.taxexemptcd + chr(3)
