oeelk.specnstype - Spec/NS
  
Label: Spec/NS
Type: character
Format: x
Decimals: ?
Initial: 
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: (S)pecial, (N)on-stock, or Blank (Regular Stock Product)
Trigger: 

Indexes:
k-specns


Description:
The special/non-stock type describes the Kit Component disposition.
It has nothing to do with the OEEL/BPEL specnstype field. If the
kit component is a non-stock product (new with Trend 7.0.000), then the
oeelk.specnstype will be "n". If the kit component is a stock product
that will be handled as a special order item, then the oeelk.specnstype
field will be "s". Otherwise the kit component will be a stock product
that will be treated as a regular item, so it will be blank. The
oeelk.specnstype will no longer have an "L" in it, if the OE line item
is lost business. The oeelk.statustype can be used in this case, since
it will contain "c" for cancelled.
 
Content:
blank = Kit component that is a stocked product
s = kit component that is a stocked product that is a specially
ordered item
n = Non-stock kit component
