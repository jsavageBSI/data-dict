icsp.msdssheetno - MSDS Sheet #
  
Label: MSDS Sheet #
Type: character
Format: x(10)
Decimals: ?
Initial: 
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
This number ties to the ICSEH records if they are set up. The sheet
number is used to record the Material Safety Data Sheet that should be
send with this product. The sheet number is NOT validated against
ICSEH, so it is possible to use the number simply as reference. It will
print on a report or on pick tickets depending on your ICAO settings.
