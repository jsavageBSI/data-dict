gdprctrl.gdprty - GDPR Type
  
Label: GDPR Type
Type: character
Format: x(1)
Decimals: ?
Initial: 
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 

Indexes:
k-gdprty


Description:
A single character code that indicates if the field specified by FIELDNM and
any associated personal data are on GDPR restriction. Disabled data is
retained in the system but it is not accessible in the application. Forgotten
data is destroyed and no longer contains information about the individual.
 
Content:
D - Disabled
F - Forgotten
Blank - No restriction
