Description:
Valid values for this field include (P)urchase, (S)ale, or (C)ustomer.
It is also possible to have (T)emp on the record which exists with a
zero claim number. This record is a controller record for all temorary
receipts for which the claim number has not yet been identified. A "T"
is not valid on any other records.
