poele.vendlineno - Vendor Line#
  
Label: Vendor Line#
Type: character
Format: x(8)
Decimals: ?
Initial: 
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: Vendor's Line Number, if the Vendor's Line# &lt;&gt; Trend Line#
Trigger: 

Indexes:
k-vendpo


Description:
If a record is being written to the EDI Audit file, the Vendor's Line
Number (from the flat file) will be written to the Audit record to
provide the line number for a cross reference for future related EDI
documents.
