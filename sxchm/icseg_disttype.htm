icseg.disttype - Distribution Type
  
Label: Distribution Type
Type: integer
Format: 9
Decimals: ?
Initial: 0
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 

Indexes:
k-icseg


Description:
Distribution type - This field defines the type of distribution to store
for balance sheet and income statements.
For balance sheets, it is set by the sasc.icglbsty field, which is entered on ICAO as: Distribute to GL Balance Sheet.
For income statments, it is set by the sasc.icglincty field which is entered on ICAO as: Distribute to GL Income Statement.
 
Content:
For income statements (icseg.accttype = false):
1-Prod Cat
2-Whse
3-Cust Type
4-Slsrep
5-Prod Cat & Whse
6-Prod Cat & Cust Type
7-Prod Cat & Sales Rep
For balance sheets (icseg.accttype = true):
1 = Prod Cat
2 = Whse
3 = Both
