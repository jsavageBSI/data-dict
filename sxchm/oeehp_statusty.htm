Initial:

Val Msg: 
Help: (A)ctive, (I)nactive or (P)artial Package

Indexes:
k-status k-whse


Description:
Package record status type
 
Content:
P = partial (when a 1/3 or a 2/3 package is written - processing does
not occur yet)
A = last package in a multipackage orders (say 3 of 3 or 1 of 1),
triggers processing for each package of the order, after which, they
are inactivated as "I" records. Also, PMES records are written back
as inactive records at this time
I = inactive
F = Final
X = Void
