vaesl.completefl - Completed Flag
  
Label: Completed Flag
Type: logical
Format: yes/no
Decimals: ?
Initial: no
Extent: 0
Mandatory: yes
Val Exp: ?
Val Msg: 
Help: 
Trigger: 

Indexes:
k-active k-prod k-specns


Description:
This flag indicates if the value add transaction section, that contains
the line item, is completed or not. This flag is part of indexes to
allow quicker access to open sections.
 
Content:
yes - the value add transaction section is completed.
no - the value add transaction section is not complete, and still open.
