Description:
The Price/Cost Per field is used to record the descriptive unit for
special pricing/costing items. If the item is not a special price/cost
item this field should be left blank. If it is a special price/cost
item, this field is required. This description will appear on most
inquiries and print outs dealing with prices or costs for this product.
Examples:
BFT = Board Foot
C = Hundred
