oeeh.pricecd - Price Level
  
Label: Price Level
Type: integer
Format: 9
Decimals: ?
Initial: 0
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: Price/Discounting Setup Customer Price Level 0 - 9 (0 = Base)
Trigger: 



Description:
The price code used for establishing the price level of the order
(defaulted from the customer record). This value is used when pricing a
line item with PDSC records. If the PDSC record is not a quantity break
record, the price code will determine the level of pricing selected from
the PDSC record. If the price code is changed in ARSC/S, the order will
not be updated. Be sure to change the price code on the order if
necessary.
 
Content:
0 through 9. 0 indicates base/list pricing.
