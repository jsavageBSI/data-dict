icscn.mincubepct - Minimum Cube Percent
  
Label: Minimum Cube Percent
Type: decimal
Format: zz9.99
Decimals: 2
Initial: 0
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
When a Carton is created, the ICSD (Warehouse) defaults setting, if
loaded for the Minimum Cube Percent will default into the Package value.
The Minimum Cube Percent cannot be greater than or equal to the
Maximum Cube Percent if the Max value is loaded.
If the Minimum Cube Percent is zero, then it is assumed that the Package
does not have a minimum cube requirement.
This in the Fill Percentage - Cube Capacity that is required to use this
package which prevents sending out a box that is not full

Not Required
