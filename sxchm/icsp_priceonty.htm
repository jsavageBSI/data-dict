icsp.priceonty - Multiplier
  
Label: Multiplier
Type: character
Format: x(1)
Decimals: ?
Initial: b
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: (B)ase price, (L)ist price or (C)ost
Trigger: 



Description:
The Price On Type indicates the standard method of pricing this product
to customers. Note that no price is indicated and this field is simply
a default. It indicates the normal multiplier method to be used
whenever selling this product. Users will usually want to set up an
initial value for this field.
If multipliers are never used, this field is irrelavent.
This field is used by PDSC to set PDSC price on type whenever a new
record is created for this product.
 
Content:
L=List Use list price as a default for multiplier pricing
B=Base Use base price as a default for multiplier pricing
C=Cost Use product cost as a default for multiplier pricing
List, Base, and Cost can be found in ICSW
