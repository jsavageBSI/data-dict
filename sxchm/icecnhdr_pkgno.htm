Initial: 0

Val Exp: ?
Val Msg: 
Help: 
Trigger: 

Indexes:
k-pkgno


Description:
This links the Package to an OEEHP record package number.
OEEHP records are created from the Shipping interface, but are numeric
numbers. Cartonization packages are based on Carton ID's.
Initially, the only way to link an external shipped package to a Cartonization
Package is manaully entering the Package Number from the Shipping
Interface to the Package in ICECN during the Shipping of the Carton - when
entering the Weight and Freigh charges.
It can be an informaitonal field that is manually entered during the shipping
process in ICECN and not be linked to OEEHP records at all.
