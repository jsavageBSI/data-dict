Description:
This field is the back order type for a drop ship line. It holds the "y" or "n" indicator from the customer since the botype is used to record that this is a drop ship line. Once the PO has been received, if the PO is back ordered, the dobotype will be set to "y". If the PO is not back ordered or if it is received in full, the dobotype will be set to no.
