icsp.dimlengthty2 - Second Dimensional Length Type
  
Label: Second Dimensional Length Type
Type: character
Format: x
Decimals: ?
Initial: I
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
If the product is set up as a Dimensional Cut product (icsp.cuttype = D), the
length of a cut piece can be entered as one number (feet) or as two
numbers (feet and inches). This field is used to indicate how the second field
should be formatted.
The length value can be entered as a whole number (Integer) or it can be
entered as a decimal number (Decimal or Fraction). The difference between
Decimal and Fraction comes when determining the relation between the first
and second parts of the length entry.
 
Content:
I - Integer
D - Decimal
F - Fraction
