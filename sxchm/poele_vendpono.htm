Label: Vendor PO #
Type: integer
Format: zzzzzz9
Decimals: ?
Initial: 0
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: Vendor's PO Number, if the Vendor's PO was &lt;&gt; to the Trend PO
Trigger: 

Indexes:
k-vendpo


Description:
This is the Purchase Order Number that was provided in the EDI 855
Inbound documents' flat file if:
- the purchase order number in the flat file cannot be found in POEH
- the purchase order number in the flat file cannot be used because
even though it was found in POEH, it matched specific conditions
that make it unusable.
In any of these cases, a new Purchase Order was created that is the pono
in this file and this field is the original purchase order number that
was in the flat file. This field will be used by additional EDI
documents so the purchase order number in the flat file for these
documents will be able to determine the corresponding purchase order in
POEH/POEL.
