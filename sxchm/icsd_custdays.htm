icsd.custdays - Customer Count
  
Label: Customer Count
Type: integer
Format: >>9
Decimals: ?
Initial: 0
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
This field indicates then number of days in which all customer owned products
in a storeroom managed warehouse must be counted. If customer owned
products should be counted once a year, then a value of 365 would
be entered. If customer owned products should be counted once a quarter, then a
value of 91 would be entered.
