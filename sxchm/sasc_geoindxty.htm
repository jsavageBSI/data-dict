sasc.geoindxty - Geo Index Type
  
Label: Geo Index Type
Type: character
Format: x
Decimals: ?
Initial: 
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: C)ity-St-Zip, St-(Z)ip, G)eoCode, Blank-Vertex
Trigger: 



Description:
Must be running Vertex to ustilize this field's settings. There is a new
SASC setting for the type of Tax Interface - data is stored in a AODATA
record (fieldname = TaxInterfaceTy). If this value is 'v', then company is
running vertex.
S - System Rates
T - Taxware
V - Vertex
GeoIndexTy field values are now limited to C, Z, G, or Blank
 
Content:
C - City-State-Zip: Run the Vertex specific logic to calculate taxes where the
city, state and zip codes must be entered before a valid GeoCode can be
loaded.
Z - Zip-State: Run the Vertex specific logic to calculate taxes where the
state and zip code must be entered before a valid GeoCode can be
loaded.
G - GeoCode: Run the Vertex specific logic to calculate taxes where the
GeoCode must be entered to load the city, state and zip code.
Blank - Vertex: Run the Vertex specific logic to calculate taxes
