Description:
The alternate warehouse if an alternate warehouse is selected. If the
OEAO alternate warehouse flag is set to yes, the operator may select an
alternate warehouse when it is displayed with substitutes and upgrades
when the ordering warehouse is short stock. Selection of the secondary
warehouse allows stock to be reserved in the alternate warehouse and
transferred (a WT is generated in OEEPI) to the selling warehouse. All
pricing/costing is per the selling warehouse. The alternate warehouse
just provides the capability to do an automatic transfer.
 
Content:
Must be valid in ICSD. The product must be valid in ICSW for both the
selling and alternaate warehouses.
