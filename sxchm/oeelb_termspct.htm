oeelb.termspct - Terms Discount %
  
Label: Terms Discount %
Type: decimal
Format: z9.99
Decimals: 2
Initial: 0
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: Terms Discount % (Valid Only if 'Terms Disc' = 'no')
Trigger: 



Description:
The terms percent can be entered at the line level to allow for special
terms discounts for a product. It defaults from PDSC or ICSP if the
SASTA terms type is line based. The net amount is accumulated in the
termslineamt at the header level for line based terms.
