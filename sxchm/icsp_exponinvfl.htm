icsp.exponinvfl - List Kit on Invoice
  
Label: List Kit on Invoice
Type: logical
Format: yes/no
Decimals: ?
Initial: no
Extent: 0
Mandatory: yes
Val Exp: ?
Val Msg: 
Help: List the Kit Components on the Invoice
Trigger: 



Description:
Explode on Invoice Print Flag. This flag is relevant only for Kit
items. If the item is a kit, this controls the invoice print routine.
 
Content:
Yes = Print all components of this kit individually on the invoice
No = Print only the kit, not the components, on the invoice
