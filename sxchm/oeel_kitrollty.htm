oeel.kitrollty - OE Kit Rollup
  
Label: OE Kit Rollup
Type: character
Format: x(1)
Decimals: ?
Initial: 
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: (C)ost, (P)rice, (B)oth or Blank
Trigger: 



Description:
The kit roll type indicates if price or cost should be rolled from the
components to the kit level.
 
Content:
blank = neither price or cost should be rolled from the components
c = only cost should be rolled
p = only price should be rolled
b = both price and cost should be rolled
The SM/GL cost for all kits (non-stock, build on demand, and prebuilt)
will ALWAYS be rolled. This field WILL NOT be checked for the SM/GL
cost. The cost roll indicator applies only to the roll of the pricing
cost and commission cost. Price will be rolled from PDSC records for the
components if they exist.
