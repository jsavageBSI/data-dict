Description:
This is a default setting that can be used per Warehouse for the creation
of new Packages for that warehouse. There is a similar AO setting that
can be used if not wanting a specific warehouse setting.
This is the dimensional unit the packages is measured in for the size of the
carton.
 
Content:
Blank - AO Default Setting
I - Inches
F - Feet
Y - Yards
CM - Centimeters
M - Meters
