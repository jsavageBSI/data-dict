Description:
The vendor from which this item is normally purchased. In the case of a
product with several vendors this should be the primary vendor. The
vendor number is required only if the arptype is 'v', but it should
always be provided. Many of the IC reports and inquiries include vendor
ranges or options that will work properly only if the vendor is provided
regardless of the ARP. This field is validated against APSV.
