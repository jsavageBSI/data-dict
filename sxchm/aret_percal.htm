aret.percal - Calendar Period
  
Label: Calendar Period
Type: integer
Format: >>>9
Decimals: ?
Initial: 0
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
This is the Calendar Period that corresponds to the fiscal period
you are currently posting in. For example if you started your fiscal
year for 1994 on July 1, 1994 for a 12 period year. Then if you made
a posting on August 5, 1994, the fiscal period would be 0294 but the
Calendar Period (aret.percal) would be 0894. The values of this
field can be from 01YY to 12YY. The field is loaded from OEET, OEEPI,
ARET, AREBU, ARER, AREMP and ARECE. It is set equal to either the
global, g-percal, or glet.percal which are derived from the GLOA
options.
