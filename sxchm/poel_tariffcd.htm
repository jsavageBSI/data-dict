Description:
Harmonized System Tariff Code. The HS system uses a logical nomenclature
method to categorize goods into approximately 5,000 commodity groups.
Countries often assign customs duties based on the harmonized tariff code
of the product.
 
Content:
A valid HS tariff code set up in SA Tariff Rates Setup (SASGT).
