icsd.cnmixbyshiptofl - Mix Multiple Orders Same Ship To
  
Label: Mix Multiple Orders Same Ship To
Type: logical
Format: yes/no
Decimals: ?
Initial: no
Extent: 0
Mandatory: yes
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
If set to no, then the order processing for cartons is by individual orders.
If set to yes, then during the cartonization process and the selection of orders
being processed during the same run, the system will see if orders can be
combined into the same cartons or not.
The criteria to combine are the same:
Customer, Whse, Ship Via, Ship To Address (1-3), Ship To City, Ship To
State, Ship To Zip Code
The order list is built off the same pick ticket run (OEEPP or WTEP) or
Order Selection in ICECN and Auto Cartonization

yes/no
