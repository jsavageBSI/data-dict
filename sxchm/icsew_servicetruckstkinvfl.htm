Description:
CSFS will use Alternate Warehouse locations as Technician's Trucks.
The system still replenishes the technician's truck like a normal warehouse,
but some SRO orders created in CSFS will need Planned items to be
loaded on their trucks for use at a job site.
Used for Line Types of 'Inventory Issue' (oeel.serviceshipty) so the
operator know's where to load the items being picked. The pick ticket
will show the whse if this field is set to yes and the AO option is set to print
for SRO orders
Some users will not utilize the Alternate Warehouse at a tech's truck

Alternate Whse is the Tech's Truck.
