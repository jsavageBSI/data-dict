oeelb.arpprodline - ARP Line
  
Label: ARP Line
Type: character
Format: x(6)
Decimals: ?
Initial: 
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
The product line or warehouse this product is sourced from. This
field is only loaded for non-stock products, special order items,
and drop ship lines. For a non-stock product, this field is
enterable on the non-stock screen. If the ARP whse on the non-stock
screen is entered, the arpprodline will contain the whse and the
arpvendno will be 0. If the ARP whse is not entered on the non-stock
screen, the arpprodline will contain the product line entered on the
non-stock screen. For a stocked product, this field defaults from either
the ARP whse (if the ARP is a whse) or the ARP product line set in ICSW.
This field is used for special and non-stock lines that need to be
picked up on a RRAR.
 
Content:
The product line (validated in ICSL) if the arp type is v.
The source warehouse (validated in ICSD) if the arp type is w/c.
