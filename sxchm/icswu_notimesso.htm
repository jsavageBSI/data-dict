icswu.notimesso - # Stockouts
  
Label: # Stockouts
Type: integer
Format: >9
Decimals: ?
Initial: 0
Extent: 25
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
Number of times stocked out - This field is set by ICAM and represents
the number of times this product was stocked out for each month in this
warehouse.
The field is a table of 25 representing each month from the current
month thru 25 months ago (in reverse order). notimesso[1] is the current
month, notimesso[2] is last month, etc.
 
Content:
D = delay reservation
R = against pending receipts
A = always
