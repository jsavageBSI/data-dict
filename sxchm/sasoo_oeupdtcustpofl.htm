sasoo.oeupdtcustpofl - Update Customer PO#
  
Label: Update Customer PO#
Type: logical
Format: yes/no
Decimals: ?
Initial: no
Extent: 0
Mandatory: yes
Val Exp: ?
Val Msg: 
Help: Allow Access to Update of Customer PO# Function
Trigger: 



Description:
Allow access to the OEET - Maintain Customer PO function. Located off
the File Menu in OEET.
Can change the Customer PO# on an order at any stage except canceled.
Used to correct PO#'s for correct Invoice Prints and billing of the customer.
 
Content:
Logical yes/no. Default is 'no' so will need to turn on the security for operators
needing access to the new function.
