Description:
Carton ID of the Package being Shipped.
See the different 'Format' options from AO and ICSD. There can be up to
four different sections to a Carton ID, or just one. The company can
determine what to use and in what position.
Sections can be:
Whse
Package Type
Carton Number
Additional Lettering
If the order/detail has an exception during Cartonization, then Carton ID
is the Order#: "o1234-00 - Exception"
