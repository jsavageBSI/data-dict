Description:
The name of the database field on the record specified in the RECORDTY
field which can hold the name of an individual. When a GDPR search is
performed from the SAAG function, Elasticsearch will look in this field to see
if the name matches the search value.
If the individulal is placed on a GDPR restriction, this field will be used to
indicate to the UI that it may need to be hidden to prevent the propagation
of the name.
