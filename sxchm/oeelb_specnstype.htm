oeelb.specnstype - Spec/NS
  
Label: Spec/NS
Type: character
Format: x
Decimals: ?
Initial: 
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: (S)pecial, (N)on-stock, (L)ost Business, or Blank
Trigger: 



Description:
The special/non-stock type describes the product/line disposition. It
can be used to identify a non-stock product, special order item, or lost
business line item. Each time the specnstype changes, OE treats the
entry as a new product and all variables are reinitialized. Handling of
the specnstype is done in oeetlns.p. Handling of the non-stock screen is
done in oeetlnus.p.
Non-Stock: the product is not stocked in this whse. All information
about the product must be manually entered.
Special order item: an "s" will be placed in the specnstype for all
Order as Needed products (since they are not stocked). A stock item may
also be specially ordered. This indicates that even if product is
available, it should not be used to fill this order. Specials and
non-stocks are picked up on the PO/WT RRARs.

is cancelled. It can be set at the line level or from a header order

blank = stocked product
n = non-stock product
s = stocked product that is a specially ordered item
l = lost business line
