notes.printallvendfl - Print All Vendors
  
Label: Print All Vendors
Type: logical
Format: yes/no
Decimals: ?
Initial: yes
Extent: 0
Mandatory: yes
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
For Product and Catalog Notes, part of the print restrictions.
If checked, the note will print on all vendor documents.
If unchecked, the restrict vendor number will be used to determine if the note should print.
 
Content:
Default is yes.
