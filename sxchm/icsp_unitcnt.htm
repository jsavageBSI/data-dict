icsp.unitcnt - Counting Units
  
Label: Counting Units
Type: character
Format: x(4)
Decimals: ?
Initial: 
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
The counting unit represents the unit of measure to be used for cycle
and physical count processes. As such, this field is used only in the
ICEP* functions.
The field is validated, if entered, from ICESU and SASTT. If not
entered, all ICEP* functions will default to the stocking unit.
 
Content:
A = active
I = inactive
L = labor
S = supercede
