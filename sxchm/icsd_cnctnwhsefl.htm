icsd.cnctnwhsefl - Use Warehouse in Carton ID
  
Label: Use Warehouse in Carton ID
Type: logical
Format: yes/no
Decimals: ?
Initial: no
Extent: 0
Mandatory: yes
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
Part of the 'Load Carton ID From Warehouse Settings' option.
If loading the Carton ID from the Whse, then this field is available to be set
for use or not.
If the selection is yes, then the warehouse is loaded into the Carton ID format
 
Content:
yes - Whse is part of the Carton ID
no - Whse is not part of the Carton ID
