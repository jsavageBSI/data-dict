oeao.oemrgholdty - Margin Hold Type
  
Label: Margin Hold Type
Type: character
Format: x
Decimals: ?
Initial: 
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: Code to be Used if OE Placed on Margin Hold
Trigger: 



Description:
This field allows the user to control what will be loaded in the Approval Type field when an Order Entry order is placed on hold based on the operator level settings.
If the margin amount or margin percentage on the Order Entry order is less than the SASO minumum margin amount or SASO minimum margin percentage respectively, the OE order will be placed on hold and the character loaded in this field will appear in the Approval Type field on the OE header. Also, if the margin amount or margin percentage on the OE order is more than the SASO maximum margin amount or SASO maximum margin percentage respectively, the OE order will be placed on hold and the character loaded in this field will appear in the Approval Type field on the OE header.
 
Content:
This field will hold a single alphabetic character used to indicate that the Order Entry order is on hold. It can be any letter except Y since that letter is used to indicate that the OE order is approved.
