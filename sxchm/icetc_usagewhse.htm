icetc.usagewhse - Usage Whse

Indexes:
k-usgwhse


Description:
Indicates the warehouse against which usage will be recorded for this
transaction. Most of the time this will match the whse where quantities
were actually affected. On occasion, the ARP will not match the
sourcing information. In this case, the usage whse will be set to the
ARP information so the usage is properly recorded against the whse that
should have provided the goods. This field is validated against ICSD
