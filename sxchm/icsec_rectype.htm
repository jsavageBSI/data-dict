icsec.rectype - Type
  
Label: Type
Type: character
Format: x(1)
Decimals: ?
Initial:

Mandatory: no

k-altprod k-custno k-icsec k-keyno k-prod


Description:
Record Type - This field represents the type of substitute product used.
 
Content:
A= Exact match, ICSP GUI ONLY (product auto-lookup)
B= Bar Code
C= Customer's product
D= PTX Vendor's Product Number (Created using PTXICSEC)
G= Catalog (ICSC)
I= Interchanges
K= GUI ONLY (product auto-lookup). Targets ICSP, ICSC to find matching records using "where CONTAINS"
M= Vendor's Product from ICSW
O= Options (see p-xref.i)
P= Supercedes
Q= GUI ONLY (product auto-lookup). Targets ICSEC, ICSP to find matching records using "where BEGINS"
S= Substitutes
T= Trade Service (Auto-Pricing)
U= Upgrades
V= Alternate Vendor Number
W= Alternate Warehouse
X= Stocked Product defined in ICSW
Y= Catalog/Customer (eSales)
Z= Catalog/Interchanges (eSales)
