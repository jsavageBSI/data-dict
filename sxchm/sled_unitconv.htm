sled.unitconv - # Stk Units
  
Label: # Stk Units
Type: decimal
Format: >>>>>9.99999
Decimals: 5
Initial: 0
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
This field is used to store the standard pack multiplier value loaded from the Supplier Link import process.
The value will be used if a new ICSEU or SASTT unit record needs to be created from the Supplier Link update process.
 
Content:
A decimal value containing the standard pack multiplier loaded from the Supplier Link data file during the import process. It may also be loaded from an existing ICSEU or SASTT unit record if no value was imported from the data file.
