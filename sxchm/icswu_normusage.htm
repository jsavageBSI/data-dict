icswu.normusage - Usage
  
Label: Usage
Type: decimal
Format: zzzzzzzz9.99-
Decimals: 2
Initial: 0
Extent: 25
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
Normal Usage - This field is updated by the ICAM run. It reflects the
number of units sold and transfered per month.
The field is a table of 25 representing each month from the current
month thru 25 months ago (in reverse order). normusage[1] is the current
month, normusage[2] is last month, etc.
