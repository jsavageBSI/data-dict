Description:
One of the specific types allowed (ZC).
It can be made to be very specific using all 10 characters of the zip code, or
it can be generic using only the first few characters (like 809). It will use the
same logic as ARSC and loading the Address when validating if the Ship To
Address zip code on the sales order applies or not:
1. Exact Match
2. First 5 characters match
3. First 3 characters match
