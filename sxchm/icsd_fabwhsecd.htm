icsd.fabwhsecd - ARP Lookup
  
Label: ARP Lookup
Type: character
Format: x(1)
Decimals: ?
Initial: 
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: VAET (P)rod Lookup on Fab Whse Use ARP Whse as Default
Trigger: 



Description:
The new field will be used to override the ?whse? default that is loaded into the Product Lookups off the VAET Line Item screen:
1. Clt-L off the Product in the browser
2. Look up off the Product on the Add New Line screen
3. Non Stock screen product lookup off the Add New Line screen
If the Warehouse on the VAET is a Fabrication warehouse, the new ICSD setting allows the whse to no longer default into the product lookup field and be insensitive. It will default in the ARP Whse (if arp set to Whse or Central Whse) and leave the Whse field on the lookup sensitive.
Allows the operator to view the availability of the other warehouse(s) to determine where to ?tie? a fabrication VA order line to.
This option cannot be set if the ICSD Whse is a Whse Logistics Whse. That is why it sits next to the WL Location field.
 
Content:
Blank for non operational
'P' to initiate the Product Lookup ARP default
