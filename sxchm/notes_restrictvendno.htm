notes.restrictvendno - Vendor #
  
Label: Vendor #
Type: decimal
Format: zzzzzzzzzzz9
Decimals: 0
Initial: 0
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
For Product and Catalog Notes, part of the print restrictions.
Used when Print All Vendors is not checked.
When the restrict vendor is entered, the note will only print on documents for that vendor.
If the restrict vendor is zero, the note will not print on any vendor documents.
 
Content:
A valid vendor # or zero.
