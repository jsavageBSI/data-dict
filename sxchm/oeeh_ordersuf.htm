oeeh.ordersuf - Order Suffix
  
Label: Order Suffix
Type: integer

Decimals: ?

Extent: 0

Val Exp: ?

Indexes:
k-custno k-jrnl k-jrnl2 k-oeeh k-whsestagecd


Description:
The order suffix of the order. The -00 always indicates the original
order. Any other order can be a back order or a release, depending on
the order type. Only a BR and jit order can have releases with suffixes
greater that -00. The borelfl indicates if the suffix is a back order or
not. Back orders are created through oeetbo.p
 
Content:
00 through 98. 99 is reserved for manual invoice entry through ARET.
