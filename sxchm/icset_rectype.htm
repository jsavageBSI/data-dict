Description:
This field describes what type of bin the count ticket was created for.
 
Content:
1 - The count was created for ICSW bin location 1
2 - The count was created for ICSW bin location 2
W - The count was created for a Warehouse Managed bin location
U - The count was created for Unavailable inventory
