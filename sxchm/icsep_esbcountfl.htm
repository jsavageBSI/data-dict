icsep.esbcountfl - ESB Count Flag
  
Label: ESB Count Flag
Type: logical
Format: yes/no
Decimals: ?
Initial: no
Extent: 0
Mandatory: yes
Val Exp: ?
Val Msg: 
Help: 
Trigger: 



Description:
This field is used to indicate if a count record has been included on a
previous Inventory Count business object document (BOD) or not.
This field only applies to a count record for a product in a storeroom
mananged warehouse.
The Inventory Count BOD is used to export count run data from SX to
the Storeroom where it will be stored in a SQL database.
 
Content:
No - the ICSEP record has not been loaded onto an Inventory Count BOD
Yes - the ICSEP record has been loaded onto an Inventory Count BOD
Default value is No
