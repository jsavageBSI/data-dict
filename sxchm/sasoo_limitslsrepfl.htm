sasoo.limitslsrepfl - Limit Sales Rep
  
Label: Limit Sales Rep
Type: logical
Format: yes/no
Decimals: ?
Initial: no
Extent: 0
Mandatory: yes
Val Exp: ?
Val Msg: 
Help: Restrict Approval Type Changes to SlsRep In/Out on the Order
Trigger: 



Description:
This logical field is used to limit the user's security to change OE approval hold codes. If set to "yes", the security applies to the user. If "no", the security does not apply. If "yes", the approval hold code field will not be sensitive in change mode if the user is not the inside or outside sales representative.
 
Content:
Valid values are "yes" and "no".
