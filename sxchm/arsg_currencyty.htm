arsg.currencyty - Currency
  
Label: Currency
Type: character
Format: x(2)
Decimals: ?
Initial: 
Extent: 0
Mandatory: no
Val Exp: ?
Val Msg: 
Help: Used Only if Group is in Foreign Currency
Trigger: 



Description:
The currency associated with the group.
The Currency Type is set up in and validated by SASTC.
If the currency type is blank, then the currency of the group is the same as
the currency of the company which is the domestic currency of the country
in which the company is located.
All customers which are set up a members of the group must have the same
currency as the group.
 
Content:
A valid currency type set up in SASTC or blank
